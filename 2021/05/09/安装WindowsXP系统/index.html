【笔记】安装 Windows XP 系统

发表于 2021-05-09 分类于 System Installation 阅读次数：

前言

在虚拟机安装WindowsXP操作系统

安装

Windows XP Professional 安装程序

  • Enter开始安装

Windows XP 许可协议

  • F8同意协议

磁盘分配

  • C创建磁盘分区

  • Enter直接默认全部磁盘空间作为磁盘分区

  • 回到磁盘分配页面按Enter将系统安装在新的分区

  • Enter直接默认NTFS分区格式

快：不包含快的选项会进行低级格式化，如果磁盘（使用很久）中有坏磁道，会自动避开坏磁道，但格式化速度很慢，如果没有使用过的新磁盘，没有坏磁道，可以选择包含快的选项

FAT：比较旧的磁盘格式

系统初始化

  • 选择语言，直接下一步

  • 设置用户，用户名必须填写（不可以是Administrator和Guest）

  • 输入产品密钥

  • 设置计算机在网络中的名称，设置管理员（Adminstrator）密码

  • 设置时间和时区

  • 设置网络，直接下一步

  • 设置工作组或计算机域，直接下一步

系统设置

  • 欢迎使用，直接下一步

  • 配置网络

  • 激活Windows

  • 注册Microsoft

  • 试用30天

由于刚刚未设置网络，所以没能与激活中心联系，此时自动跳转为试用30天

  • 设置用户

  • 设置完成
