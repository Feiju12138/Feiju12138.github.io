前言

Node.js 是能够在服务器端运行 JavaScript 的开放源代码、跨平台 JavaScript 运行环境。Node.js 由Node.js Foundation（已与JS Foundation合并为OpenJS Foundation）持有和维护，亦为 Linux 基金会的项目。Node.js采用Google开发的V8运行代码，使用事件驱动、非阻塞和异步输入输出模型等技术来提高性能，可优化应用程序的传输量和规模。这些技术通常用于资料密集的即时应用程序。（维基百科）

模块（module）

  • 当一个.js文件可以独立完成一个功能的时候，这个.js文件就是一个模块
  • 当一些.js文件可以共同完成一个功能的时候，这些.js文件就是一个模块

暴露

  • Nodejs中js文件作用域隔离，因为Nodejs没有window对象
  • 如果我们需要让js文件之间相互通信，需要让js文件自己暴露

exports

  • 将file.js文件内的变量暴露出来，在app.js调用暴露出来的变量

module.exports

  • 将People.js的构造函数暴露出来，在app.js通过People.js暴露出来的构造函数创建对象

People.js

app.js

如果要暴露多个参数，使用exports
如果只暴露一个参数，并且是构造函数，使用module.exports

exports与module.exports

  • exports是module.exports的一个引用
  • module.exports可以直接赋值，它将修改暴露的本质（暴露一个对象），也就是说暴露的数据类型是什么，得到的数据数据类型就是什么
  • exports直接赋值，不会修改暴露的本质（还是一个对象），也就是说不管暴露的时候是什么数据类型，得到的数据数据类型还是对象，如果暴露的时候不是对象，那么得到的将是一个空对象
