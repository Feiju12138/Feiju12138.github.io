Keywords

  • Reserved words in C++
  • Keywords cannot be used as variable names or constant names

Identifier Naming Rules

  • The identifier cannot be a keyword
  • The identifier can only consist of letters, numbers, and underscores
  • The first character cannot be a number
  • Identifiers are case-sensitive

Data Type Identifier Memory Space Value Range

  • Used to represent decimal numbers
  • The significant figures include both digits before and after the decimal point

Operators

Arithmetic Operators

  • + (positive sign), - (negative sign), + (addition), - (subtraction), * (multiplication), / (division), % (modulus), ++ (increment), -- (decrement)

Note:
The result of dividing two integers is still an integer, and the decimal part will be discarded.
Because 0 cannot be used as the divisor in mathematics, the dividend cannot be 0 in both division and modulus operations.
Increment and decrement can be divided into prefix and postfix.

  • Prefix: increment or decrement first, then perform the calculation.
  • Postfix: perform the calculation first, then increment or decrement.

Assignment Operators

  • = (assignment), += (add and assignment), -= (subtract and assignment), *= (multiply and assignment), /= (divide and assignment), %= (modulus and assignment)

Comparison Operators

  • Comparison operators yield boolean values after calculation

  • == (equal to), != (not equal to), < (less than), > (greater than), <= (less than or equal to), >= (greater than or equal to)

Logical Operators

  • && (logical AND), || (logical OR), ! (logical NOT)
