Preface

Learning notes on generic programming in C++

Function Templates

Define Function Template

  • Define a generic data type
  • typename can be used for function templates, class can be used for class templates, they can also be mixed

T: generic data type, usually written in uppercase, commonly named as T

  • First define the template, followed by the function definition, this forms the definition of a function template

Calling Function Templates

Automatic Type Deduction

  • Automatic type deduction requires consistent data types when passing multiple parameters
  • No implicit type conversion occurs

Explicitly Specify Type

  • If the function does not use custom types, then only explicitly specifying the type can call the function template
  • The explicit type needs to be written inside <>
  • Implicit type conversion can occur

Regular Function and Function Template

  • When regular functions and function templates coexist, the regular function is called first
  • If you want to force the use of a function template, you can use <> to implement an empty template parameter list

Overloading Function Templates

  • Overloading method implementation using specialized types can be called first

Class Templates

Define Class Templates

  • Define a generic data type
  • typename can be used for function templates, class can be used for class templates, they can also be mixed

T: generic data type, usually written in uppercase, commonly named as T

  • First define the template, followed by the class definition, this forms the definition of a class template

Create Objects by Calling Class Templates

  • Class templates do not have automatic type deduction
  • Member functions of class templates are not called at the beginning, but are created when objects are created

Default Values in Class Templates

  • The template parameter list of class templates can have default values

Timing of Calling Member Functions in Class Templates

  • Member functions in class templates are not created during compilation, but when called
