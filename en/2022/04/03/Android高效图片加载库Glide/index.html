Preface

Glide is a fast and efficient open source media management and image loading framework for Android that wraps media decoding, memory and disk caching, and resource pooling into a simple and easy to use interface. (Github)

Import Glide Dependency

Add Network Permission

  • Add network permission outside the <application></application> tag in the manifest file

Add Image View in Layout

Render Image Using Glide in Java

Render Image Directly

  • Render a network image into an ImageView

<url>: URL of the network image resource

Use Placeholder
  • You can configure multiple placeholder images at the same time, or only configure one placeholder image

.placeholder(): Placeholder image to display when the image is being requested
.error(): Placeholder image to display when the request fails. If not set, the placeholder configured by .placeholder() will be displayed by default
.fallback(): Placeholder image to display when the request is empty. If not set, the placeholder configured by .placeholder() will be displayed by default
.override(<width>, <height>): Specify the width and height of the placeholder image in pixels. If not specified, it will follow the configuration of the ImageView

.apply(requestOptions): Apply the configuration

Transition Animation for Placeholder

  • Transition animation when transitioning from the placeholder to the actual image

  • Glide provides a default cross fade effect that can be used directly

  • Using the default Drawable method

.transition(DrawableTransitionOptions.withCrossFade(<time>)): Specify the duration of the transition animation

  • Using the Bitmap method

.transition(BitmapTransitionOptions.withCrossFade(<time>)): Specify the duration of the transition animation

Image Transformation
  • When using image transformation, it only works for .jpeg format images
Circle
Rounded Corner
  • All four corners are rounded based on the specified radius

<reg>: Radius of the corners

  • Each of the four corners has a separate radius

<reg_left_top>: Radius of the top left corner
<reg_right_top>: Radius of the top right corner
<reg_right_bottom>: Radius of the bottom right corner
<reg_left_bottom>: Radius of the bottom left corner

Rotation
  • Rotate the image clockwise by the specified angle

<reg>: Angle of rotation

Glide Extensions

  • Using Glide annotation processor
  • Use annotations to generate a more efficient Glide class on a class level

Import Glide Annotation Processor Dependency

Use GlideApp Class

Create Configuration Class

  • Create a MyGlideModule class (custom class name) as the configuration class, extending the AppGlideModule class, and add the @GlideModule annotation on the class to generate the GlideApp class
  • Recompile the project after creating the configuration class to generate the GlideApp class

Use GlideApp Class to Add Placeholder Configuration

  • With the GlideApp class, you can use a fluid API to add placeholder configurations without creating a RequestOptions object

Configure Using Frequently Used Options Set

Create Configuration Class

  • Create a MyGlideExtension class (custom class name) as the configuration class, and add the @GlideExtension annotation
  • Must create a private parameterless constructor
  • Extract frequently used configurations as option sets, create static methods that return the option sets, and add the @GlideOption annotation on the methods
    • The configuration set method names can be customized
    • Multiple configuration set methods can be specified
    • Transformation effects cannot be configured in the configuration set
  • Recompile the project after creating the configuration class to make the configuration set method effective

Use Configuration Set

  • Using the configuration set can improve the efficiency of creating objects

defaultOption(): Configuration set method name
