Priority

BEGIN: Indicates what to do before processing the data stream
PROGRAMME: Indicates what to do when processing the data stream
END: Indicates what to do after processing the data stream

Output Statement

Output specified column (field)

$0: Print all columns of each line
$N: Print the Nth column of each line
$NF: Print the last column of each line

Output specified line (record)

N: Specify the line

Specify column separator

  • The default column separator is whitespace or tab
  • You can use -F to modify the column separator

Output string

  • Add a string in the output statement

Separate multiple groups of data

  • Use , to separate multiple groups of data, and the output will be separated by a space
  • Equivalent to adding a space string
