【英文】MacOS上部署嵌入式开发环境

Posted on 2022-10-19 Edited on 2024-03-28 Views:

Preface

Compiling C source code for 51 single-chip microcontrollers on MacOS and downloading the compiled code to the microcontroller. This article is valid for both Intel and Apple Silicon versions of Mac.

Preparation

Install Compiler

  • sdcc

Install Driver

  • CH34xVCPDriver.pkg

  • After successful installation, you can verify through the ls /dev/tty.* or ls /dev/cu.* commands

The suffixes for each person are usually different. For example, the suffix for my first Mac Thunderbolt interface is 10, and the suffix for my second Mac Thunderbolt interface is 110.

Install Downloader

  • The downloader grigorig/stcgal is a module written in Python, so Python environment is required. Install using Pip

Code Writing

  • The source code of the microcontroller is written in C language. The C language code editor is omitted here.
  • C51 code written in Windows uses Keil compiler, and C51 code written in Mac uses sdcc compiler.
  • The values for defining the IO interface in Keil cannot use binary numbers (0b00000000), only hexadecimal numbers (0x00), while the sdcc compiler can use binary numbers (0b00000000).
Remark Windows Keil Statement or Keyword Mac sdcc Statement or Keyword
Header File Definition reg51.h、reg52.h 8051.h、8052.h
Definition of IO Ports sbit P0^0; __sbit P0_0;
Default Variable Definition for IO Ports P2^0 P2_0
Interrupt Function void xxx() interrupt 1 {...}; void xxx() __interrupt 1 {...};
Default Storage Type Used in Small Storage Mode, Corresponding to Direct Addressing data __data
Default Storage Type Used in Medium Storage Mode, Corresponding to 8-bit Register Indirect Addressing pdata __pdata
Default Storage Type Used in Large Storage Mode, Corresponding to 16-bit Register Indirect Addressing xdata __xdata
With Indirect Addressing, You Can Access the High 128 Bits of ROM idata __idata
Store data in ROM, can only store constants unsigned char code xxx[] = {...}; __code unsigned char xxx[] = {...};
Definition of bit variable. The bit addressing area of 51 is in the 0x20~0x2f area of the built-in RAM, with a total of 128 bits bit __bit
Definition of special function register sfr P0; __sfr P0;

About the use of the at keyword

  • Specifies the storage address of the variable to reduce the use of pointers
Remark Windows Keil Statement or Keyword Mac sdcc Statement or Keyword
Use address when defining IO ports sbit P0^0 = 0x81; __sbit __at (0x81) P0_0;
Use address when defining special function register sfr P0 = 0x80; __sfr __at (0x80) P0;

Compilation

  • Compile to obtain the .ihx file, .ihx can be further converted to .hex file and .bin file

Direct Compilation

Compile into a Library First, Then Complete the Compilation

  • Usually used for large projects with multiple modules. Compile all modules into library files first, then compile all library files together
  • When some modules are modified, only need to compile the modified modules into new library files first, then combine the library files of other modules with the new library files of the modified modules for compilation

<source>: C language source code file

<source_1>: source code that has not changed, usually compiled into a library link library
<source_2>: source code that has changed, usually compiled together with the library file compiled from the source code that has not changed
<library>: link library file

Convert ihx file to hex file (Optional)

<file>: file name

Convert ihx file to bin file (Optional)

Files Generated After Compilation

<file>: compiled file name

Download

  • Make sure that the driver is installed, the computer is connected to the microcontroller and there are no problems with the test

-P auto: specify the type of the microcontroller

auto: automatically determine

-p Port: specify the connected port

/dev/cu.wchusbserial10: If the microcontroller is connected to the first interface of Mac, the suffix is usually 10
/dev/cu.wchusbserial110: If the microcontroller is connected to the second interface of Mac, the suffix is usually 110

<file>.ihx: compiled file
