Posted on 2022-10-24 Edited on 2024-03-28 In Java后端学习指北 Views:

Introduction

Feign is a Java to HTTP client binder inspired by Retrofit, JAXRS-2.0, and WebSocket. Feign’s first goal was reducing the complexity of binding Denominator uniformly to HTTP APIs regardless of ReSTfulness. (Github)

Add Dependencies

Add Annotation in Main Application Class

  • Add @EnableFeignClients annotation in the main application class to enable Feign auto-configuration.

Manually Specify Client Class

Create Interface

  • Create an interface for sending requests
  • Feign annotations are the same as those in Spring MVC

@FeignClient(""): specifies the request address (or service name)
@GetMapping(""): sends a Get request and specifies the request parameters
@PathVariable: specifies the request parameters in Rest style

Send Requests Using Feign

  • Use Feign instead of RestTemplate to send requests

Modify Log Level

Using Configuration File

feign.client.config.default.loggerLevel: sets the log level for all services

NONE: default value. No logs.
BASIC: basic information. Includes request send time, request end time, and request duration.
HEADERS: detailed information. Includes request headers and response headers.
FULL: complete information. Includes request body and response body.

feign.client.config.serviceName.loggerLevel: sets the log level for a specific service

Using Configuration Class

Create Configuration Class

Configure Global Application

  • Add parameters to the annotation in the main application class

Configure Specific Service

  • Add parameters to the annotation in the FeignClient interface

Performance Optimization

  • Replace the default non-connection-pooling implementation URLConnection with a connection-pooling implementation Apache HttpClient

Add Dependency

Configure Connection Pool

feign.httpclient.enabled: enable Feign’s support for HttpClient
feign.httpclient.max-connections: maximum number of connections
feign.httpclient.max-connections-per-route: maximum number of connections per route
feign.httpclient.connection-timeout: request timeout
feign.httpclient.time-to-live: connection’s time to live

Best Practices

  • Extract Feign as a service. When other services need to use the Feign client, inherit the Feign service to use it directly.
