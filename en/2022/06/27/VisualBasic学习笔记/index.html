Introduction

Visual Basic (VB) is an event-driven programming language developed by Microsoft. It is derived from the BASIC programming language. VB features a graphical user interface (GUI) and a rapid application development (RAD) system, which allows for easy use of database connectivity through DAO, RDO, ADO, or the creation of ActiveX controls. Programmers can easily build an application using the components provided by VB. (Wikipedia)

Identifiers

  • Identifiers must start with a letter or Chinese character, followed by letters, Chinese characters, numbers, or underscores
  • The last character of a variable name can be a type character
  • VB is case-insensitive
  • Identifiers cannot have the same name as keywords

Keywords

  • Keywords, also known as reserved words, are predefined identifiers with special meanings in a programming language

Constants

  • In VB, there are two types of constants: literal constants and symbolic constants
  • Literal constants are represented by their actual values
  • Symbolic constants are represented by an identifier that replaces a constant value
    • Predefined symbolic constants within the system
    • User-defined symbolic constants

Operators and Expressions

Arithmetic Operators

Precedence Operator Note
0 () Highest precedence
1 ^ Exponentiation
2 - Negation
3 * Multiplication
3 / Division
4 \ Integer division
5 Mod Modulo
6 + Addition
6 - Subtraction
  • Integer division discards the decimal part and only keeps the integer part of the quotient
  • The sign of the modulus is determined by the sign of the number before the modulo operator
  • If subtracting two date values, the result will be the number of days between the two dates
  • Adding or subtracting an integer from a date value still results in a date value

Relational Operators

Operator Note
= Equal to
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to
<> Not equal to
Like Comparison with wildcards
Is
  • When comparing numerical values, the comparison is based on their numerical values
  • When comparing letters, the comparison is based on the ASCII code values of the letters
  • When comparing Chinese characters, the comparison is based on the ASCII code values of the first pinyin letter of the characters
  • When comparing Chinese and English, Chinese is considered greater than English because the Unicode encoding for Chinese is higher than that of English
  • Relational operators all have the same precedence

Like

Wildcard Note
* Matches any number of characters
? Matches any single character
# Matches any single digit

Logical Operators

Precedence Operator Note
Highest () Highest precedence
High Not Negation. Returns the logical complement
Medium And Logical AND. Returns False if any expression is False; returns True if all expressions are True
Low Or Logical OR. Returns True if any expression is True; returns False if all expressions are False
