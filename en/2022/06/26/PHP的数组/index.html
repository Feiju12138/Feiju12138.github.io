Introduction

Notes on PHP arrays

Defining an Array

  • Arrays can store data of any data type. If arrays store arrays, they become multi-dimensional arrays
  • In PHP, arrays are stored in key-value pairs. If no key is specified, the key defaults to the array index

Defining Arrays Using Language Structures

Defining an Indexed Array

  • Data is added to the array with the same array name, by default, it is appended to the end of the array. If no key is specified, the key is the index of the data, starting from 0

Defining an Associative Array

  • Data is added to the array with the same array name, by default, it is appended to the end of the array

Mix Definition of Arrays

  • If adding data with and without keys simultaneously, the keys of data without keys still start from 0

Defining Arrays Using Functions

Defining an Indexed Array

  • The key is the index of the array

<value>: Data stored in the array, can be of any data type

Defining an Associative Array

  • Stores key-value pairs, keys and values are linked by =>

<key>: Key of the data, usually a string
<value>: Value of the data

Mix Definition of Arrays

  • If adding data with and without keys simultaneously, the keys of data without keys still start from 0

Operations on Data Stored in Arrays

Add

  • Data is directly added to the array with the same name

Update

  • Data with the same key name in the array with the same name is directly modified
    • For an indexed array, pass the index value
    • For an associative array, pass the key name

Retrieve

  • Call the array with the same name
    • For an indexed array, pass the index value
    • For an associative array, pass the key name
