【英文】通过UUP的方式下载Windows官方安装镜像

Posted on 2022-02-05 Edited on 2024-03-28 Views:

Introduction

Download Windows official installation image through UUP (Unified Update Platform)
UUP is Microsoft’s way of referencing system updates on Windows 10, using incremental updates to reduce file download size

Preparation

Dependencies required on Mac

Get the internal version number of the Windows image

  • https://changewindows.org/

Download Build Package

  • Visit https://uupdump.net/, search by internal version number, and choose the desired build

Cumulative Update: Cumulative update version, integrates patches directly into the system
Feature Update: Feature update version, official release of major version + patches, patches will be installed after system installation
Insider Preview: Insider Preview version

  • Select language-> Next

  • Choose the system version according to your needs

  • Select download method-> Create Download Package

Download method: If you only need the system version you just selected, simply select Download and Convert using aria, if you need to package more versions together, you can choose Download, convert using aria2, and create other versions, and select the desired additional versions in Select other versions
Build options: If using a non-Windows platform, the first three options are invalid

Download Dependencies on Mac platform (optional)

Start building

  • After downloading and extracting the build script, enter the directory, and start building
