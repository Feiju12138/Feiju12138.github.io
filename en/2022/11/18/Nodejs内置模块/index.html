Nodejs built-in module study notes

fs file system module

  • When using relative paths, you may encounter dynamic concatenation errors
    • Principle: when using relative paths, the current path is concatenated with the relative path to form an absolute path, and then read/write the file through the absolute path
    • Dynamic concatenation error: If the current path is not in the same directory as the code path, the dynamically concatenated absolute path will be incorrect

Import Module

Read File Content

  • If the read is successful, err is null and dataStr is the content of the file read
  • If the read fails, err is an error object, and dataStr is undefined

<file>: the file path to read
utf-8: specify character encoding

Write File Content

  • If the file name being written already exists, it will overwrite the original file
  • When writing a file, only files can be created, not directories. It will throw an error if the directory does not exist.

<file>: the file path to write
<text>: the content to write
utf-8: specify character encoding, can be omitted, the default is utf-8
