    • 定时器模式（工作方式1-5）：计数初值从THx和TLx（x为0或1）的初始值开始减到0

    • 计数器模式（工作方式0-4）：计数初值从THx和TLx（x为0或1）的初始值开始加到最大值

    • 对应关系：

    Mode 工作方式 描述
    0 0 13位定时器/计数器
    1 1 16位定时器/计数器
    2 2 8位自动重装定时器/计数器
    3 3 忽略
    4 0（手动） 13位定时器/计数器
    5 1（手动） 16位定时器/计数器
    6 2（手动） 8位自动重装定时器/计数器
    7 3（手动） 忽略

    写入计数初值

    • 通过THx和TLx（x为0或1）寄存器写入计数初值
    • 这两个寄存器在不同的模式下有不同的功能
    定时器模式
    • 定时器工作方式是除以12的机器周期
    • 定时器周期 = 计算机器周期 * (THx << 8 + TLx)
    计数器模式
    • 初始值 = 0
    • 计数 = THx << 8 + TLx

    开启定时/计数器

    • 通过TR0和TR1（x为0或1）寄存器控制开启和关闭定时/计数器
    • TRx寄存器是可位寻址的
    • TR0和TR1分别对应定时/计数器0和定时/计数器1
    • 0：关闭定时/计数器
    • 1：开启定时/计数器
