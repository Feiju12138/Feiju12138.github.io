Preface

Proteus 8 Professional learning notes

Basic Operations

Zooming Canvas

  • Zoom In: Scroll Up
  • Zoom Out: Scroll Down

Dragging Canvas

  • Click Scroll Wheel to start dragging the canvas, click again to cancel dragging canvas

Placing Components

  • After selecting a component, click the left mouse button on the canvas for the first time, and you can continue to select the release position
    • Click the left mouse button again to confirm the release
    • Click the right mouse button to cancel the release
    • Press the “+” key on the numeric keypad to rotate clockwise 90 degrees, and press the “-“ key to rotate counterclockwise 90 degrees

Deleting Components

  • Double-click the right mouse button on the component to be deleted to quickly delete the component
  • Select the component to be deleted, right-click the mouse, and select “Delete Object”

Editing Component Properties

  • Double-click on a component that has been placed on the canvas

Component Wiring

  • Click one end of a component, and then click one end of another component to connect them
    • Double-click one end of a component to repeat the previous wiring method

Auto Wiring

  • When placing a component, simply place the pins directly opposite each other to automatically connect them

Aligning a Group of Components Quickly

  • Select a group of components and press Ctrl+A to automatically align them

Modifying Settings

Modifying Canvas Size

  • System(alt+Y) -> Set Paper Size(S)

Modifying Grid Size

  • View -> Modify Grid Size

The smaller the grid, the more precise the adjustment of component positions

Modifying Shortcuts

  • System(alt+Y) -> Set Shortcuts(K)

Drawing Basic Circuit Components

  • Select Terminal Mode in the sidebar

Terminals with the same name are connected to each other

Drawing Components from the Library

Adding Components from the Library

  • Open the component chooser

Common Components and Configurations

  • All components that need to be simulated should not be selected from the “DEVICE” library, but from the “ACTIVE” library
  • All software that does not need to be simulated can be checked “Don’t Simulate”

Single Chip Microcomputer: AT89C52

Program File: Specify the compiled .hex program of the microcontroller
Clock Frequency: Specify the clock frequency. The clock circuit connected to the microcontroller is invalid, and this configuration is used instead

Crystal: CRYSTAL

Frequency: Specify the crystal frequency

Non-polar Capacitor: CAP

Resistance: Specify the capacitance value

Polar Capacitor: CAP-POL

Resistance: Specify the capacitance value

Button: BUTTON
Resistor: RES

Resistance: Specify the resistance value
Model Type: Specify the mode

ANALOG: Analog mode
DIGITAL: Digital mode. Digital mode should be used in the microcontroller environment; in real life, there is no distinction between modes

Resistor Array: RESPACK-8
Light-Emitting Diode (Yellow): LED-YELLOW

Forward Voltage: Specify the turn-on voltage
Full drive current: Specify the maximum operating current

7-Segment Display (4-digit/Blue): 7SEG-MPX4-CA-BLUE (The 8 pins on the left are digit pins, and the 4 pins on the right are segment pins)
Active Buzzer: BUZZER
Passive Buzzer: SOUNDER
Speaker: SPEAKER
Dot Matrix Display (8x8): MATRIX-8X8-GREEN (The 8 pins on top are row pins, and the 8 pins on the bottom are column pins)

Drawing Graphics

Selecting a Shape to Draw

Creating a Configuration (Optional)

  • Specify a new configuration name

  • Edit Configuration

The width of the text cannot exceed 50

Drawing Shapes

  • Select a configuration and draw shapes
    • If you are drawing text, you also need to set the text content and properties

Inserting Images

  • File -> Insert Image

Batch Modifying Component Properties

  • Select the “Property Assignment Tool” in the shortcut toolbar

String: The attribute name on the left of the equal sign represents the attribute to be assigned, and the value on the right of the equal sign represents the value to be assigned

REF=XXX: Specify the reference attribute of the component
VALUE=XXX: Specify the Resistance attribute of the component
NET=XXX: Specify the string attribute of the pin
XXX=#: If the attribute value contains ordered variables, the # can be used as a placeholder, and the initial value and step size of the variable can be controlled by Counter Initial Value and Counter Increment. If the counter increment is positive, the variable increases; if the counter increment is negative, the variable decreases

Operation: The operation to be executed
Apply To: If you need to manually select the components to be batch assigned, you can select Click Object. If a group of components has been selected, you can choose Current Selected Object or Global Selected Object. If you need to batch assign values to all components, you can choose All Object

Starting Simulation

  • If the microcontroller has specified program code, the code can be executed

  • Debug -> Start Simulation

    • Or click the start button in the lower-left corner

Pitfalls

  • Error occurred when opening the library: No Libraries Found!

Cause

  • The installation is in the C drive, and the permission is insufficient to open the library

Solution

  • Run Proteus as an administrator by right-clicking and selecting “Run as Administrator”
