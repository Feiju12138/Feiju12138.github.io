Operations on Excel Applications

Start a New Application

  • Requires installation of Microsoft Office Excel

Parameters

visible=None: Whether the application is visible

True: Visible
False: Not visible
spec=None: Specify the location of the Excel program
For Windows, specify the default Excel opening method in settings
For Mac, you can directly pass the path of the Excel application here
add_book=True: Whether to automatically create a workbook when opening the application
True: Automatically create a workbook
False: Do not automatically create a workbook
