Preface

C++ deque container study notes
The deque container is also called a double-ended array
The data structure of deque allows insertion or deletion at both ends
The deque is internally maintained by a controller, which stores elements in a buffer, so the data structure of deque is not a continuous memory space, but a continuous space
Deque access elements are not as fast as vectors
