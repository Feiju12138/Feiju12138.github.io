Introduction

Notes on Excel functions

Composition of Functions

Constant

  • Composed of numbers directly

Variable

  • Composed of cell references

Cell Variable

Relative Reference

  • Composed of column number and row number
  • Relative references automatically change when filling cells

Absolute Reference

  • Add $ before the column number or row number
  • Absolute references do not change when filling cells

Mixed Reference

  • Variables that contain both relative and absolute references

  • When autofilling, only the row changes automatically

    • Typically when filling cells vertically
  • When autofilling, only the column changes automatically
    • Typically when filling cells horizontally

Find Data Based on Keywords

Search by Rows
  • Using a keyword to search for rows that meet the keyword conditions and return data from other columns in that row

Keyword: Select a cell as the search keyword
Search range: Select a group of cells as the search range
Offset of the column to return the result: Which column in the found row should be returned as the result, counting from the column where the keyword exists as the first column
Matching criteria

0 or True: Exact match
1 or False: Fuzzy match

Fuzzy Match Example

Search by Columns
  • Using a keyword to search for columns that meet the keyword conditions and return data from other rows in that column

Keyword: Select a cell as the search keyword
Search range: Select a group of cells as the search range
Offset of the row to return the result: Which row in the found column should be returned as the result, counting from the row where the keyword exists as the first row
Matching criteria

0 or True: Exact match
1 or False: Fuzzy match
