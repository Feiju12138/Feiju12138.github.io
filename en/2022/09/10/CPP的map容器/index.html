【英文】C++的map容器

Posted on 2022-09-10 Edited on 2024-03-28 In C++学习指北 Views:

Foreword

Notes on using the map container and multimap container in C++.
All elements in a map are pairs. The first value of the pair is called the key, which is used for indexing; the second value is called the value, which is used for storing elements.
map and multimap are associative containers, which automatically sort the elements upon insertion.
The data structure of map and multimap is a binary tree.
Duplicates of keys are not allowed in a map, so it can be used for deduplication.
Duplicates of keys are allowed in a multimap.
