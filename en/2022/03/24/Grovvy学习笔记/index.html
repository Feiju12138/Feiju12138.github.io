Introduction

Apache Groovy is an object-oriented programming language designed for the Java platform. This dynamic language has some features similar to Python, Ruby, and Smalltalk, and can be used as a scripting language for the Java platform. Groovy code is dynamically compiled into Java bytecode running on the Java Virtual Machine (JVM), and it can interoperate with other Java code and libraries. Due to its JVM-based nature, Groovy can utilize libraries written in other Java languages. Groovy’s syntax is very similar to Java, and most Java code also complies with Groovy’s syntax rules, although there may be semantic differences. Groovy 1.0 was released on January 2, 2007, and Groovy 2.0 was released in July 2012. Starting from version 2, Groovy can also be statically compiled, providing type inference and performance similar to Java. Groovy 2.4 is the last major version sponsored by Pivotal Software and was released in March 2015. Groovy has changed its governance structure to the Project Management Committee (PMC) of the Apache Software Foundation. (Wikipedia)
