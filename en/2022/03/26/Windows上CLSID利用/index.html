Preface

Use CLSID to create shortcuts for system programs, which can be used to hide private folders.

Using CLSID to Create System Program Shortcuts

  • Rename the normal folder, add .{CLSID} as the suffix, and the folder will be disguised as a shortcut for a system program. The files inside the folder will not be lost, and you can use the command prompt or remove the suffix .{CLSID} from the file name to operate the files.

Using CLSID to Remove and Restore Explorer Sidebar Icons

Remove System Icons

  • In the registry editor, find the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\ folder and the HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\ folder. Delete the corresponding folder based on the {CLSID} to remove the corresponding system icon.

Remove 3D Objects

  • In the registry editor, find the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\ folder and delete the {0DB7E03F-FC29-4DC6-9020-FF41B59E513A} folder.

Remove OneDriver

  • In the registry editor, find the HKEY_CLASSES_ROOT\CLSID\{018D5C66-4533-4307-9B53-224DE2ED1FE6} folder and change the value of System.IsPinnedToNameSpaceTree from 1 to 0.

Remove Dropbox

  • In the registry editor, find the HKEY_CURRENT_USER\Software\Classes\WOW6432Node\CLSID\{E31EA727-12ED-4702-820C-4B6445F28E1A}\ShellFolder folder and change the value of Attributes from f080004d to f090004d.

Remove CreativeCloudFiles

  • In the registry editor, find the \HKEY_CLASSES_ROOT\CLSID\{0E270DAA-1BE6-48F2-AC49-80DEEFDBB87F} folder and change the value of System.IsPinnedToNameSpaceTree from 1 to 0.

Restore System Icons

  • In the registry editor, find the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\ folder. Right-click on the NameSpace folder and click New->Key. Use the {CLSID} as the value to restore the corresponding system icon.

General Usage of CLSID

  • Directly use the command shell:::{CLSID} to start the corresponding system program.
