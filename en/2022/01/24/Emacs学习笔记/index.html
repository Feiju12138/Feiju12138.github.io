Introduction

Emacs (/ˈiːmæks/, derived from Editor MACroS, macro editor) is a family of text editors known for their extensibility, with users primarily being programmers and other computer users working in technical fields. It was originally completed by Richard Stallman and Guy L. Steele Jr. at MIT in 1975. The idea for Emacs was inspired by the TECO macro editor TECMAC and TMACS, which were macro text editors written by Guy L. Steele Jr., Dave Moon, Richard Greenblatt, Charles Frankston, and others. (Wikipedia)

Command Annotations

Basic Commands

Page Navigation

Scroll down

  • C-v

Scroll up

  • M-v

Move the cursor to the center of the screen

  • C-l

Cursor Control

Move up/down

  • C-p(revious)
  • C-n(ext)

Move left/right by one character

  • C-b(ackward)
  • C-f(orward)

Move left/right by one word

In English, it’s by word; in Chinese, it’s by sentence separated by punctuation marks

  • M-b(ackward)
  • M-f(orward)

Move to the beginning/end of the line

  • C-a
  • C-e

Move to the beginning/end of the sentence

  • M-a
  • M-e

Move to the beginning/end of the file

  • M-S-,
  • M-S-.

Show the current line as the beginning/middle/end line of the page

  • C-l

Specify the number of times a command is executed

  • C-u number shortcut

Text Operations

Editing text

  • Enter text directly

Delete one character before/after the cursor

  • delete
  • C-d

Delete one word before/after the cursor

  • M-delete
  • M-d

Cut all characters from the cursor to the end of the line/sentence

  • C-k(ill)
  • M-k(ill)

Cut all selected characters

  • C-S-2 select C-w

Paste the most recently cut text

  • C-y

Paste previously cut text

  • C-y M-y

Undo

  • C-/
  • C-S–
  • C-x u

File Operations

Find and edit a file

If the file is not found, it will be created

  • C-x C-f(ind) file path

List all buffers

  • C-x C-b

Exit buffer list

To close all other windows and keep only one

  • C-x 1

Switch buffers

  • C-x b buffer name

Save current/multiple buffers

  • C-x C-s(ave)
  • C-x s

Quit Emacs

  • C-x C-c(lose)

Restore autosaved files

  • Mx recover file

Search strings downwards/upwards

  • C-s(earch) keyword
  • C-r keyword

Panel Operations

Keep only one panel

Close all other windows and keep only one window

  • C-x 1

Split the current content into 2 panels (horizontal arrangement)

  • C-x 2

Split the current content into 2 panels (vertical arrangement)

  • C-x 3

Open a new panel

Open a new window and specify a file

  • C-x 4 C-f file path

Scroll down in other panels

  • C-M-v

Move the cursor to other panels

  • C-x o(other)

Window Operations

Create a window

  • M-x make-frame

Close a window

If the last window is closed, Emacs will be closed

  • M-x delete-frame

Abort all operations

  • C-g
  • Esc Esc Esc
