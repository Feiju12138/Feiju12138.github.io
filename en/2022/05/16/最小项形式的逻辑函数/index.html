【英文】最小项形式的逻辑函数

Posted on 2022-05-16 Edited on 2024-03-28 Views:

Introduction

Notes on learning logic functions in minimal form

Comparison Karnaugh Map

Pattern

  • First, fill in the 16 squares with numbers 0-15

  • Then, swap the rows in area A

  • Then, swap the columns in area C

  • The final complete part is the 16-square Karnaugh Map
  • The upper half is the 8-square Karnaugh Map

Type 1

  • Use Karnaugh Map method to convert the logic function to minimal form

Example: Convert the logic function L=AB+BCD to minimal form using Karnaugh Map method

  1. Represent the function in the Karnaugh Map

  1. Compare the Karnaugh Map in the question with the comparison Karnaugh Map

Based on the comparison, find the entries that are 1 in the Karnaugh Map in the question, and find the corresponding entries in the comparison Karnaugh Map. Express the result using mnumber


L=AB+BCD=m1+m4+m5+m7+m6+m9

Type 2

  • Use formula method to convert the logic function to minimal form

Example: Convert the logic function L=AB+AC to minimal form using formula method

  1. Use the formula: A(B+B)=A, make sure each term contains all letters A, B, C (or A, B, C, D), and simplify it in the order of A, B, C (or A, B, C, D)

L=AB+AC
L=AB(C+C)+AC(B+B)
L=ABC+ABC+ACB+ACB
L=ABC+ABC+ABC+A BC

  1. Replace A, B, C, D with 1, A, B, C

111 110 011 001

  1. Convert the obtained numbers from binary to decimal

7 6 3 1

  1. Use the obtained numbers as subscripts for m

∴ L=m7+m6+m3+m1

Type 3

  • Convert the minimal form to variable form

Example: Convert L(A,B,C)=m7+m6+m3+m1 to variable form

  1. Extract the subscript numbers

7 6 3 1

  1. Convert the obtained numbers from decimal to binary, if L(A,B,C) has three letters, convert them to 3-bit binary (or 4-bit binary for four letters).

111 110 011 001

  1. The first bit represents A, the second bit represents B, the third bit represents C (or the fourth bit represents D), connect them with OR, and obtain a set of logic functions.
    • If the bit is 1, it represents the letter itself.
    • If the bit is 0, it represents the negation of the letter.

L=ABC+ABC+ABC+ABC

  1. Simplify the logic function using Karnaugh Map method or formula method

L=AB+AC

Type 4

  • Simplify the expression in the form L=Σm+Σd using Karnaugh Map

Example: Simplify L(A,B,C,D)=Σm(1,4,5,6,7,9)+Σd(10,11,12,13,14,15) using Karnaugh Map

  1. Determine whether an 8-square Karnaugh Map or a 16-square Karnaugh Map should be used based on the number of letters in L(A,B,C,D)
  • If there are 3 letters, use the 8-square Karnaugh Map
  • If there are 4 letters, use the 16-square Karnaugh Map
  1. In the comparison Karnaugh Map, change the squares corresponding to the positions of m(1,4,5,6,7,9) in the expression to 1, change the squares corresponding to the positions of d(10,11,12,13,14,15) to ×, and change the squares that are not modified to 0

  1. Treat × as 1, find 8 1’s (or 4 1’s, 2 1’s, 1 1’s) and express all 1’s. It is not necessary to express all ×. Obtain the logical expression.

L=B+CD
