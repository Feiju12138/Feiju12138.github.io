Preface

Karnaugh Map Study Notes

Classification of Karnaugh Map

  • Karnaugh maps can be divided into 8-cell and 16-cell maps.

The 8-cell map has three regions: A, B, and C.

The 16-cell map has four regions: A, B, C, and D.

Note: Whether it is an 8-cell or 16-cell map, the positions of regions A, B, C, and D on the Karnaugh map are fixed.

Type 1

  • Represent the Karnaugh map as a logical expression L (taking the 16-cell map as an example in this case).

  1. Find 8 consecutive 1s.
    • If 8 consecutive 1s cannot be found, find 4 consecutive 1s.
    • If 4 consecutive 1s cannot be found, find 2 consecutive 1s.
    • If 2 consecutive 1s cannot be found, find 1 consecutive 1.

Consecutive means that all the 1 cells can form a rectangle.



Note: The four boundaries (including the four corners) can be considered as capable of forming a rectangle.


  1. Determine whether the found 1s are all in, all out, or partially in regions A, B, C, or D.

∴ Partially in region A
∴ Partially in region B
∴ Fully in region C
∴ Partially in region D

  1. Perform logical AND on the conclusion codes.
    • If they are all inside, write the codes directly.
    • If they are all outside, add an underline to the codes.
    • If they are partially inside, do not write the codes.

∴ C

  1. Exclude the previously used 1s
    • If there are still unused 1s, repeat steps 1-3, and step 1 thereafter needs to meet the requirement: include as many (at least 1) unused 1s as possible.
    • If there are no unused 1s left, terminate the loop.

∴ C
∴ AD
∴ ABD

  1. Finally, perform logical OR on all the results.

∴ C+AD+ABD

Type 2

  • Given a logical expression L, draw the Karnaugh map.

  1. Determine whether the result is an 8-cell or 16-cell map based on the number of variables, and draw all the empty cells.

  2. Divide the logical expression L by the plus sign (+) and segment them.

  3. For each segment in the current iteration, determine the regions for each variable and set the overlapping region to 1.

  4. Repeat steps 2-3 until all segments are completed.

  5. Finally, set the remaining cells to 0.

Type 3

  • Given a logical expression L, simplify it using Karnaugh map.

  1. Draw the Karnaugh map based on the given logical expression.

  2. Simplify the obtained Karnaugh map into a new logical expression.

Type 4

  • Represent the Karnaugh map as the complement function L of the logical expression L.

  • Method 1

    1. First, find the logical expression L using the Karnaugh map.
    2. Then, find the complement function L of the logical expression L.

  • Method 2

    1. Change the position of 1s to 0s and 0s to 1s in the Karnaugh map.
    2. Find the logical expression L of the original Karnaugh map using the new Karnaugh map, and the result is the complement function L of the logical expression L.
