【英文】Defaults学习笔记

Posted on 2022-05-13 Edited on 2024-03-28 Views:

Preface

The defaults command can be used to modify the configuration files (.plist) of all Mac applications.

Configuration File Path

  • The default is to modify the configuration files in the user’s directory.

View Global Identifiers of Software

  • View the global identifiers of system software or user software that can be directly modified using the defaults command.

Syntax

Read Configuration

<appname>: The global identifier of the system application.

Modify Configuration

<key>: Specify the key.
<type>: Specify the type of value.
<value>: Specify the value.

Reset to Default Configuration

Delete Configuration

  • After the configuration is deleted, it will be restored to the default configuration.

Configuration of Program Dock

Delete all configurations

Lock Content

Modify Configuration

<value>

true: Lock
false: Unlock

Delete Configuration

Configuration Position

Modify Configuration

<value>

left: On the left side of the screen
right: On the right side of the screen

Delete Configuration

Lock Position

Modify Configuration

<value>

true: Lock
false: Unlock

Delete Configuration

Configuration Icon Size

Modify Configuration

<value>: The default value is 48.

Delete Configuration

Lock Icon Size

Modify Configuration

<value>

true: Lock
false: Unlock

Delete Configuration

Configure Automatic Hide

Modify Configuration

<value>

true: Enable automatic hide
false: Disable automatic hide

Delete Configuration

Configure Automatic Hide Animation Time

Modify Configuration

<value>: The default value is 0.7

Delete Configuration

Configure Automatic Hide Trigger Delay

Modify Configuration

<value>: The default value is 0.5

Delete Configuration

Configure Whether to Show Recent Application Icons

Modify Configuration

<value>

true: Show recent application icons
false: Hide recent application icons

Delete Configuration

Configure Minimization Animation Effects

Modify Configuration

<value>

genie: Genie effect
scale: Scale effect
suck: Suck effect (hide effect)

Delete Configuration

Configure Whether Hidden Application Icons Are Translucent

Modify Configuration

<value>

true: Hidden application icons become translucent
false: Hidden application icons do not become translucent

Delete Configuration

Configure Whether to Enable Bouncing Animation for Newly Opened Applications

Modify Configuration

<value>

true: Enable bouncing animation for newly opened applications
false: Disable bouncing animation for newly opened applications

Delete Configuration

Configure Animation Time for Triggering Corners

Modify Configuration

<value>: Animation time, if set to 0, means no animation

Delete Configuration
