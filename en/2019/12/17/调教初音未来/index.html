Preface

Hatsune Miku is a virtual female singer developed and sold by Crypton Future Media based on Yamaha’s VOCALOID 2 voice synthesis engine. It is the first work in the character vocal series and is a VSTi-featured electronic instrument. It was released on August 31, 2007, at an open price, and the official estimated price of the software series is about 15,750 Japanese yen. (Wikipedia)

Preparation

Software Preparation

  • Hatsune Miku VOCALOID voice library (Chinese version)

  • Logic Pro X (App Store)

For Windows users, “Fruit Studio” or “YAMAHA VOCLOID” can be used as alternatives.

  • Piapro Studio (included with the purchase of Hatsune Miku)

  • Chinese to Pinyin Converter (App Store)

  • Microsoft Word (App Store)

Download Original Version of the Song

  • It is recommended to support official versions. Since I am not using it for commercial purposes and only using it for temporary tutorials, I directly obtained the song from the browser’s cache.

Download Lyrics

Pinyin Annotation for Lyrics

  • Use software to annotate the lyrics with Pinyin. Only save the Pinyin, as there is no software that can directly use the “Chinese to Pinyin Converter” webpage tool for conversion.

Remove Pinyin Annotation

  • Open Microsoft Word and copy all the Pinyin with annotations to a new page.

  • Use the “find and replace” function to remove all line breaks first.

  • Find the line break code “^p” and change it to a space “ “.

  • Find the Pinyin with annotations and replace it with lowercase English letters without Pinyin:

ā á ǎ à -> a
ē é ě è -> e
ī í ǐ ì -> i
ō ó ǒ ò -> o
ū ú ǔ ù -> u

Download Sheet Music

Import Midi File into Piapro Studio

  • First import the Midi file into Logic Pro X.

  • There might be other tracks included. Export the desired track as a Midi file.

  • Import the new Midi file into Piapro Studio and align the vocals with the Midi.

  • However, after aligning, it is discovered that the downloaded Midi file from the internet does not match the desired vocal part, so a manual creation of the vocal Midi file is required.

Manually Create Vocal Part Midi File

  • Create a new project in Logic Pro X.

  • Set the track to a piano sound and configure the settings to match the tempo, time signature, and key of the original song or sheet music.

  • Import the song material and instrumental material into Logic Pro X. Find the starting point of the vocal part and start composing on the piano roll according to the sheet music.

  • The number “1” in the simplified notation corresponds to the piano keyboard key “C3,” and so on.

  • Sample of the composed melody
  • Export the composed Midi file.

Tune Hatsune Miku in Piapro Studio

  • Import the composed Midi file into Piapro Studio and adjust the starting position of the vocal part.

  • Double-click on Hatsune Miku’s track to zoom in and double-click on the vocal block to input lyrics.

  • Sample of lyrics input

Export

  • If detailed tuning is needed, continue tuning.

  • After tuning is complete, export as audio.

  • Format conversion (optional)
