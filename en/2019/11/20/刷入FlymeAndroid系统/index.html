【英文】刷入 Flyme Android 系统

Posted on 2019-11-20 Edited on 2024-03-28 Views:

Preface

Flyme is an Android-based mobile operating system developed by Meizu Technology Co., Ltd. (Wikipedia)

The phone I use is a second-hand “Samsung Galaxy S5 (China Mobile customized version with Samsung system)”. The data cable is “Gongniu Android data cable”. The computer system is Windows and MacOS. After flashing the phone, the system is “Flyme 6.7.12.29R beta (Meizu system)”, and the initial and final Android versions are “Android 6.0.1”.

Warning: This article is only for sharing successful cases, and any problems that occur will be at your own risk.

Preparation

  • SAMSUNG_USB_Driver_for_Mobile_Phones.exe (Samsung mobile driver for Windows)

  • SmartSwitch4Mac_setup.dmg (Samsung mobile S switch assistant for Mac)

  • samsung_scl23_scl23_mmb29m.scl23kdu1dqa4_recovery.tar.md5 (Samsung original Recovery)

  • twrp-3.0.2-1-kltechn.img.tar (TWRP third-party Recovery)

  • flyme_SM-G9008W_ddfzl_6.7.12.29R-beta.zip (Flyme Samsung S5 version Meizu system ROM package)

  • SDK Platform Tools (adb tool package)

  • Samsung SD card (if card brush)

Begin Flashing

Install drivers on Windows and MacOS systems respectively

  • Windows

  • MacOS

Unlock BootLock

Skip this step because Samsung S5 does not have BL lock enabled by default.

Flash third-party Recovery (TWRP)

Phone enters Fastboot mode

  • Press “Volume-“+”Home”+”Power” buttons

  • Press “Volume+” to confirm

Flash third-party TWRP

  • Open “Odin 3” software on Windows system

  • Connect the phone to the computer with a data cable

  • Odin, the first column, the third row changes color, indicating successful connection

  • Click on “AP” and select the TWRP third-party Recovery, then click “Start”

This process may fail. If it fails, please flash back to the original Samsung Recovery.

  • When the progress bar on the phone is full, the phone will automatically restart

  • Hold down “Volume+”+”Home”+”Power” buttons to enter Recovery mode when rebooting

  • The first time you enter TWRP, you need to set the language to “Chinese (Simplified)”

Flash the system

“Wipe clean”

  • Select “Wipe”

  • Select “Advanced Wipe Menu”

  • Select all folders except “SD Card” and “USB Storage”

  • Slide to confirm deletion

  • Return to the lower left corner

  • Select “Format Data Partition”

  • Enter “yes” to confirm

“Card” brush (optional)

  • Put the Flyme system image in the root directory of the SD card

  • In the TWRP homepage, select “Install”

  • Select “Storage” -> “SD Card”

  • Select “.zip” Flyme system image -> “Slide to confirm flashing”

“Wire” brush

  • Mac uses HomeBrew to download “android-platform-tools”
  • In the TWRP homepage, select “Advanced”

  • Select “ADB Sideload”

  • Use the terminal command to brush

Completion

  • After successful flashing, the phone will start up very slowly for the first time.

  • Show a photo of the completed flashing as a keepsake.
