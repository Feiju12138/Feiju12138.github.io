【英文】安装黑苹果系统（MacOS Catalina）

Posted on 2019-11-15 Edited on 2024-03-28 Views:

Introduction

The Apple system installed on Apple computers is called the “White Apple” system, while the Apple system installed on non-Apple computers is called the “Black Apple” system. This section will explain how to install the Black Apple system.

Since each computer model is different, if you encounter any problems, you should copy and paste the log code to search on Baidu.

Prepare a Windows PE maintenance USB drive for formatting the entire disk or system recovery in the event of any unexpected situations.

Preparation

  • Computer with an Intel processor

  • USB drive with a capacity of 8GB or more (for creating the Black Apple system installation disk)

  • Black Apple system image (lazy version)

  • Computer with Windows system that can be accessed

  • TransMac

  • USB drive (for creating the Windows PE system maintenance USB drive)

Creating the System Maintenance USB Drive (optional)

Download the DA bai cai USB Startup Disk Creation Tool

  • Go to the DA bai cai official website on a Windows system

  • Download the DA bai cai USB Startup Disk Creation Tool (UEFI version is recommended)

Install the DA bai cai USB Startup Disk Creation Tool

  • Right-click on the “DA bai cai” installation package and run it as an administrator

  • Click “Install” to start the installation

  • Wait for the installation to complete and click “Experience now”

Use DA bai cai to create the USB Startup Disk

  • Insert the USB drive that you want to create the USB Startup Disk with into the computer

  • Select the recently inserted USB drive in the “Select Device” section

  • Click “Start” to begin the creation process

Completion of creating the USB Startup Disk

Creating the Black Apple System Installation Disk

Download the Lazy Version Black Apple System Image

  • The so-called lazy version image is a fusion of the Apple original system and Clover Bootloader

Here is the latest MacOS Catalina Black Apple system image created by the respected 黑果小兵 here

Install the Tool to Create the Black Apple Installation Disk

  • Install the TransMac Tool on a Windows system

Install the TransMac Tool

  • Right-click on the “TransMac” installation package and run it as an administrator

  • Next

  • Select “I agree to the agreement” -> Next

  • Set the installation location -> Next

  • Set the folder name for the “Start” -> Next

  • Create a desktop shortcut -> Next

  • Start the installation -> Install

  • Installation complete -> uncheck “View Readme.txt” -> Finish

Use the TransMac Tool to create the Black Apple installation disk

  • After opening TransMac, a prompt will appear asking if you want to purchase the tool. Since it will only be used once, the 14-day trial is sufficient. Wait for the 10-second countdown and click “Run”

  • In TransMac, select the USB drive that you want to create the Black Apple installation disk on. Right-click -> “Format Disk for Mac” to format it in the Mac file storage format

  • After formatting is complete, select the USB drive again and right-click -> “Restore with Disk Image” to create the Black Apple installation disk

  • After selecting the Black Apple image, wait for the installation to complete

Completion of creating the Black Apple installation disk

Preparing for Installation

Accessing “MacOS Utilities”

  • Insert the Black Apple installation disk and quickly press the “F12” key to enter the manual selection of the startup option before the system starts. (Different computer models have different hotkeys for accessing the startup options)

  • Select the Black Apple installation disk

  • The system will load Clover Bootloader for the first time

  • On the Clover Bootloader page, select the hard drive with the keyword “Install” and press enter to confirm

  • The system will load MacOS Recovery for the second time

  • Access “MacOS Utilities” and select “Simplified Chinese”

Formatting the Disk

  • Select “Disk Utility” -> Continue

  • Select the hard drive where you want to install the Black Apple system -> Erase

You can name it whatever you like, I named it “Macintosh HD” (because that is the default hard drive name for the White Apple version), and choose the “APFS” format

  • Click “Erase”, wait for completion, and click “Done”

  • Exit Disk Utility

Installing the System Package on the Hard Drive

  • Select “Install MacOS” -> Continue

  • Continue to confirm

  • Agree

  • “I have read and agree to the terms of the software license agreement” -> Agree

  • Select the previously formatted hard drive -> Install

  • Wait -> All installation data will be transferred to the hard drive at this time

Official Installation

System automatically installs

  • At this time, the computer will restart automatically, run Clover Bootloader again, and select the hard drive with the keyword “Macintosh HD”

  • Run Clover Bootloader again, enter the automatic installation page, and the installation will proceed automatically without any manual operation

System Initialization

Afterword

  • So far, this Black Apple system cannot boot without a USB drive that has Clover Bootloader. Some functions cannot be used due to the lack of driver installation (such as no audio driver, so no sound can be played), and there are a few bugs (such as sleep mode freezing). It is very troublesome to find drivers for different hardware for Black Apple systems, and I don’t know how to write them myself, so I will continue to improve the Black Apple system when I have the opportunity.
