Preface

Hypertext Markup Language is a standard markup language used for creating web pages. HTML is a fundamental technology that is often used together with CSS and JavaScript by many websites to design user interfaces for web pages, web applications, and mobile applications. Web browsers can read HTML files and render them into visual web pages. HTML describes the structural semantics of a website along with the presentation of its contents, making it a markup language rather than a programming language. Wikipedia

Syntax

  • File extensions for HTML documents:

.html
.htm

  • Tags can be divided into:

Double tags: They have both opening and closing tags
Single tags: The opening and closing tags are combined

  • Tags can be nested

  • Attributes can be defined within the opening tags, where attributes are made up of key-value pairs and the values need to be enclosed in quotation marks

  • HTML tags are not case-sensitive, but it is recommended to use lowercase

Code

Attributes

Modifying Elements with Attributes

  • Attribute declarations must be placed within opening tags

  • Multiple attributes can be added to a single tag, separated by spaces

  • The order of multiple attributes does not matter

Every Attribute Has a Value

  • Attributes and their values are connected with =

  • Attribute values are enclosed in quotation marks

Standard Attributes

  • Each tag has its own specific attributes

  • Attributes that are supported by most tags are called standard attributes

id="": Identifier
title="": Title
class="": Class name
style="": Style

Metadata Tags
  • Metadata tags are placed within the head tag. The information contained in the head tag is generally not accessible to users in browsers, but for search engines, they usually focus on grabbing the contents of the head tag when crawling web pages.

charset="utf-8": Character set of the webpage
name="keywords" content="": Keywords of the webpage
name="description" content="": Description of the webpage
name="author" content="": Author of the webpage
http-equiv="refresh" content="5; http://localhost/": Countdown of 5 seconds to automatically redirect to another webpage

Link Tags

href="": Link address, with attribute values:

href="#": Return to the current page
href="#id": Anchor link found using the id
href="#name": Anchor link found using the name

target="": Window name
title="": Link tooltip text

Image Tags

Supported formats: JPEG, PNG, GIF
src="": Path and filename
alt="": Alternate text for invalid image
width="": Width
height="": Height
border="": Adds a specified width external border to the image. The default color is black and the default width is 2px
title: Tooltip text to be displayed when the mouse is hovered over the image

Video Tags

width="": Width, default width is 300px
height="": Height, default height is 150px
src="": Video path
type="video/mp4": Type, only supports mp4, ogg, and webM formats
controls: Display controls
poster="": Set cover image path
autoplay: Automatically play (Chrome browser requires a default mute for this to take effect)
muted: Default mute
loop: Loop play automatically

Audio Tags

src="": Audio path
type="audio/mp3": Type, only supports mp3, ogg, and wav formats
controls: Display controls
autoplay: Automatically play (Chrome browser requires a default mute for this to take effect)
muted: Default mute
loop: Loop play automatically

List Tags

Ordered List

type="a": Sort with lowercase letters
type="A": Sort with uppercase letters
type="1": Sort with numbers
start="2": Start sorting from 2

Unordered List

type="disc": Solid circle
type="circle": Hollow circle
type="square": Solid square

Definition List

  • In a definition list, the dl tag defines the list, the dt tag defines the title, and the dd tag defines the content.
  • Multiple dd tags can be described under a single dt tag, and multiple dt tags can be used.
