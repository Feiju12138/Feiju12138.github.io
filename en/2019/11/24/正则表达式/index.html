Introduction

Regular expression is a concept in computer science. It uses a single string to describe and match a series of strings that comply with a certain syntactic rule. In many text editors, regular expressions are commonly used to search for and replace text that matches a certain pattern. Wikipedia

Basic Syntax

Define matching characters that meet certain conditions

Specify a specific character or string

Exact match
  • Must be the character a
Or match
  • Must be the string ab or ac

Specify an uncertain character

  • Any of the characters a, b, or c should be included
Specify a character
Specify a range

Define characters that do not meet the conditions (negation)

  • Any character that does not include any of the characters a, b, or c is acceptable

Specify times

  • Immediately following the character, it means the number of times the previous character is allowed to appear

Specify a specific number of times

Not specifying a specific character

Specify the number of times as a range
Interval
  • 1 to 2 times
Greater than or equal to
  • 1 or more times
Specify the number of times as 1 or 0
  • It can have or not have, equivalent to {0,1}
Specify the number of times as 1 or more
  • Equivalent to {1,}
Specify the number of times as 0 or more
  • Matches any number of times, equivalent to {0,}

Wildcard

Any whitespace character

  • Including newline, space, and tab

Any non-whitespace character

  • Excluding newline, space, and tab

A digit character

  • Equivalent to [0-9]

A non-digit character

  • Equivalent to [^0-9]

A digit, letter, or underscore character

  • Equivalent to [0-9a-zA-Z_]

A non-digit, letter, or underscore character

  • Equivalent to [^0-9a-zA-Z_]
