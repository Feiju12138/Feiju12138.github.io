Introduction

Homebrew is a free and open-source software package management system designed to simplify the process of installing software on macOS. It was initially written by Max Howell. (Wikipedia)

Install Xcode Command Line Tools

  • xcode command line tools

Install HomeBrew

  • Visit the brew official website to check for the latest update code

  • Copy the latest installation code to the terminal

  • Confirm by pressing Enter

  • Wait for the installation to be successful

After installation, check the version of the software to confirm a successful installation

Pitfalls

  • Error prompt Failed to connection ...

Just paste the installation code a few more times, usually pasting for the third time will move to the next step successfully

  • Error prompt Password:

Request for password input, may require input multiple times, since the installation directory is a system directory, it requires admin permission. Alternatively, you can add sudo before the installation code to avoid entering the code multiple times

  • Even if Xcode command line tools are not manually installed, they will be automatically installed after executing the brew installation command

Add Environment Variables

  • MacOS systems with M1 chip need to add homebrew to environment variables for it to work
