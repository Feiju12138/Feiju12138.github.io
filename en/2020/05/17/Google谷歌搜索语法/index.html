【英文】Google谷歌搜索语法

Posted on 2020-05-17 Edited on 2024-03-28 Views:

Introduction

Notes on advanced syntax for Google search engine.

Basic Operators

Exact Match

  • "keyword",''

Fuzzy Match

  • *keyword*

Logical Operators

Logical AND

  • "keyword"+"keyword"
  • "keyword"AND"keyword"

Logical OR

  • "keyword"|"keyword"
  • "keyword"OR"keyword"

Logical NOT

  • "keyword"-"excluded keyword"
  • "keyword"NOT"excluded keyword"

Advanced Operators

Search Specific Site

  • site:keyword

Search Specific URL

  • inurl:keyword

Search Specific Site Title

  • intitle:keyword

Search Specific Site Content

  • intext:keyword

Search Specific File Type

  • filetype:file type
  • ext:file type

Retrieve Cache

  • cache:keyword

Search Links of a Website

  • link:keyword

Find Basic Information of a Specific Site

  • info:keyword

Examples

Link

Search PHP Website Backend

  • inurl:admin.php login

Search Discuz! Template Forum Sites

  • intext:" power by discuz! 7.2" inurl:faq.php intitle:forum
  • intext:Powered by Discuz

Search WP Template Sites

  • intext:powered by woordpress

Search CMS Template Sites

  • intext:Powered by *CMS

Search Nessus Vulnerability Scan Results

  • intitle:"Nessus Scan Report" ext:.html

Search Cacti Monitoring Pages

  • intitle:"Cacit" AND inurl:"/monitor/monitor.php"

Search File Indexing Sites

Search for file indexing sites containing Index of /

  • "Parent Directory"

Search Pages Vulnerable to SQL Injection

  • inurl:login
  • inurl:"*.php?id=" "You have an error in your SQL syntax"
  • "You have an error in your SQL syntax near"

Search phpinfo Pages

  • intext:"PHP Version " ext:php intext:"disable" intext:"Build Date" intext:"System" intext:"allow_url_fopen"
