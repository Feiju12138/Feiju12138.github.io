Dynamic SQL Statements

  • After enabling dynamic SQL statement configuration, using dynamic SQL statements can help prepare SQL objects and prevent SQL injection

Execute SQL Statements

  • DDL: Operations on databases and tables for CRUD
  • DML: Operations on data for CRUD
  • DQL: Query data

<sql>: SQL statement

Execute DDL Operations

  • Returns the number of rows successfully executed, returns 0 if execution fails
  • It can be operations on tables or on the database

Execute DML Operations

  • Returns the number of rows successfully executed, returns 0 if execution fails, also returns 0 for deleting databases, usually as long as the DDL statement does not throw an error, it is considered executed successfully

Execute Query Statements

  • Returns a result set
