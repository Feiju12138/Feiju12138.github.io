Python is a widely used interpreted, high-level, general-purpose programming language. Python supports multiple programming paradigms, including functional, imperative, structured, object-oriented, and reflective programming. Variable definition

Variable definition rules

  • The first character can only be a letter or an underscore (using an underscore is not recommended)
  • Subsequent characters can only be letters, digits, or underscores
  • Case sensitive

Naming conventions

  • Meaningful
  • Short
  • All lowercase
  • Separate multiple words with underscores
  • Variables are generally named with nouns, while functions use predicates (verb + noun)
  • Class names start with an uppercase letter and use camel case

Variable assignment and operations

Variable assignment

  • Use = to assign a value to a variable

  • In Python, variables do not need to be declared with a type before first use, but they must be initialized when they are defined to determine the variable type

Variable operations

Standard operators

  • +, -, *, /, //, %, **

/: The division operator represents division. The result of division between integers is of type integer, while the result of division between an integer and a float is of type float
//: Floor division represents integer division. The result is the smaller integer generated by dividing two approximately close integers. The result type is determined by the dividend, either integer or float
%: Modulo operation, returns the remainder of the division of two numbers
**: Exponentiation, returns the result of raising the first number to the power of the second number

  • Python also supports +=, -=, *=, /=
  • However, Python does not support ++ and -- operations. ++ and -- placed before a variable represent positive and negative sign operations, and placing them after a variable will result in an error

Comparison operators

  • <, <=, >, >=, ==, !=, <>

Both != and <> are not equal operators, but != is commonly used for not equal

Chained comparison

  • Python supports chained comparison

  • A < B > C is equivalent to A < B and B > C

Logical operators

  • Arranged according to priority: not, and, or

  • The result is either True or False (capitalized)

Data types

Numeric types

int: Integer
long: Long integer
bool: Boolean, with values True and False corresponding to 1 and 0
float: Floating-point number

complex: Complex number, usually only used for mathematical operations

Number representation

  • Python displays numbers in decimal by default
  • Numbers that start with 0 indicate octal numbers
  • Numbers that start with 0x or 0X indicate hexadecimal numbers
  • Numbers that start with 0b or 0B indicate binary numbers

Strings

  • A collection of characters between quotes ('' or "")

  • A combination of a pair of three (single or double) quotes that preserves the format of the strings

  • Python does not distinguish between characters and strings
