Introduction

Basic operations on documents in Linux

cat command

  • Display contents of a small document

<option>: option
<file>: file path and name

less command

  • Display contents of a large document in pages

Basic operations of less

Scroll down by line

  • return key
  • ↓ key
  • j key
  • control+j key
  • control+n key

Scroll up by line

  • ↑ key
  • k key
  • control+k key

Scroll down by page

  • space key
  • f key
  • control+f key

Scroll down by half a page

  • d key
  • control+d key

Scroll up by page

  • b key
  • control+b key

Scroll up by half a page

  • u key
  • control+u key

Go to the beginning of the document

  • g key
  • control+g key

Go to the end of the document

  • G key
  • control+G key

Search

<value>: keyword

  • /<value>

Next occurrence

  • n key

Previous occurrence

  • N key

Edit document

  • Open current document in the default editor

  • v key

  • control+v key

Display less help document

  • h key
  • control+h key

Display only lines with specified string

<str>: specified string

  • ZZ key
