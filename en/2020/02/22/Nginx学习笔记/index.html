Server Status Codes

Common

Status Code Meaning Note
301 Moved Permanently The requested resource has been permanently moved to a new location
302 Found The requested resource is temporarily available at a different URI
305 Use Proxy The requested resource must be accessed through the proxy specified in the location
307 Temporary Redirect The requested resource is temporarily available at a different URL
400 Bad Request
402 Payment Required Reserved for future use, for some digital currencies or micropayment
403 Forbidden The server understood the request, but refused to execute it
404 Not Found Request failed, resource does not exist
406 Not Acceptable The server cannot generate a response that is acceptable by the request’s headers

Uncommon

Status Code Meaning Note
408 Request Timeout
409 Conflict The request could not be completed due to a conflict with the current state of the resource
410 Gone The requested resource is no longer available and no forwarding address is known
413 Payload Too Large The server is refusing to process the request because the request entity is larger than the server is willing or able to process
417 Expectation Failed The server cannot meet the requirements of the Expect request-header field
418 I’m a teapot Hyper Text Coffee Pot Control Protocol, but not implemented by actual HTTP servers
420 Method Failure
422 Unprocessable Entity The request was well-formed but was unable to be followed because of semantic errors
500 Internal Server Error The server encountered an unexpected condition which prevented it from fulfilling the request
502 Bad Gateway
504 Gateway Timeout The request was successful, but the server stopped responding
