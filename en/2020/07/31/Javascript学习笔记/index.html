Preface

JavaScript (usually abbreviated as JS) is a high-level, interpreted programming language. JavaScript is a prototype-based, function-first language that is multi-paradigm, supporting object-oriented, imperative, and functional programming. It provides syntax for manipulating text, arrays, dates, and regular expressions, but does not support I/O, such as networking, storage, and graphics, which can be provided by its host environment. It has been standardized by ECMA (European Computer Manufacturers Association) through ECMAScript. It is used by the vast majority of websites in the world and supported by the world’s leading browsers (Chrome, IE, Firefox, Safari, Opera). (Wikipedia)

Data Types

  • Basic data types: String, Number, Boolean, Null, Undefined
  • Reference data type: Object

Checking the current data type

string

    • Strings in JS need to be enclosed in quotes.
  • Double or single quotes can be used, but do not mix them.
  • Quotes cannot be nested; double quotes cannot contain single quotes, and vice versa.
  • When using typeof to check a string, it will return string.

Number

  • In JS, all numbers are of the Number type.
  • This includes integers and floating-point numbers.
  • When using typeof to check a number, it will return number.

Maximum Value

  • If a number represented by Number exceeds the maximum value, it will return Infinity (for positive infinity).

  • If a number represented by Number is below the minimum value, it will return -Infinity (for negative infinity).

  • Using typeof to check Infinity will also return Number.

  • Integer operations in JS are generally accurate.

  • If float operations are performed in JS, an inaccurate result may be obtained, so do not use JS for operations requiring high precision.

Boolean type

  • Boolean values are either true or false and are mainly used for logical evaluation.
  • true represents true, false represents false.
  • When checking a boolean value with typeof, it will return boolean.

Null type

  • The Null (null) type value is specifically used to represent an empty object.
  • When checking a null value with typeof, it will return object.

Undefined type

  • The Undefined (undefined) type value is only one, which is undefined.
  • When a variable is declared without being assigned a value, its value is undefined.
  • When checking an undefined value with typeof, it will return undefined.

Type conversion

  • Type conversion refers to forcibly converting one data type to another.
  • Type conversion mainly refers to converting to String, Number, and Boolean.

Converting other data types to String

toString
  • Call the toString() method of the converted data type.
  • This method does not affect the original variable and will return the converted result.
  • Note: null and undefined do not have a toString() method. If you call their methods, it will throw an error.
String()
  • Call the String() function and pass the converted data as an argument to the function.
  • When using String() for type conversion:
  • For Number and Boolean, it calls the toString() method.
  • For null and undefined, it does not call the toString() method. It directly converts null to “null” and undefined to “undefined”.

Converting other data types to Number

Number()
Converting strings to numbers
  • If it is a pure number string, it is directly converted to a number.
  • If the string contains non-numeric content, it is converted to NaN.
  • If the string is an empty string or a string consisting entirely of spaces, it is converted to 0.
Converting Booleans to numbers
  • true is converted to 1.
  • false is converted to 0.
Converting Null to a number
  • Result is 0.
Converting Undefined to a number
  • Result is NaN.
parseInt()

  • parseInt() function: Converts a string to an integer.

  • parseInt() can extract valid integer content from a string.

  • parseFloat() function: Converts a string to a floating-point number.

  • parseFloat() functions similarly to parseInt(), but with support for valid decimals.

  • If parseInt() or parseFloat() is used on a non-String, it will first convert it to a String and then operate.

Converting other data types to Boolean

  • Using the Boolean() function.
  • Numbers are converted to Boolean values, where anything except 0 and NaN is true.
  • Strings are converted to Boolean values; anything other than an empty string is true.
  • Null and undefined are converted to false.

Others

  • In JS, to use hexadecimal numbers, they need to start with 0x.
  • To represent octal numbers, they need to start with 0.
  • To represent binary numbers, they need to start with 0b.
  • However, not all browsers support this feature.

Operators

Arithmetic operators: +, -, *, /, %, ++, –
Assignment operators: =, +=, -=, *=, /=, %=
Comparison operators: ==, !=, >, >=, <, <=
Bitwise operators: &, |
Logical operators: &&, || (false && expression, true || expression)
Logical negation operator: ! (not)
Ternary operator: expression ? expression : expression
