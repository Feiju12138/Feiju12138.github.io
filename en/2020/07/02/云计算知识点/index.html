Preface

Cloud computing knowledge, Linux basics

Code

Check memory size and free space

  • cat /proc/meminfo

View interpreter

  • cat /etc/shells

View directory contents

  • ls

  • View directory contents in long format

    • ls -l

  • Display hidden content

    • ls -a

  • Provide readable capacity units

    • ls -h

  • Display detailed information about the current directory itself

    • ls -ld

  • Display detailed information with capacity units

    • ls -lh

View CPU

  • lscpu

View file contents

  • cat

  • Display line numbers

    • cat -n

View system version

  • cat /etc/redhat-release

View IP address information

  • ifconfig

    • eth0

      • The first network card

    • lo

      • Loopback address, IP address is always 127.0.0.1

    • virbr0

      • Virtualization network card

Restart the system

  • reboot

Change hostname (temporary)

  • hostname xxx

Network Manager (change hostname) (graphical interface)

  • nmtui

Create a new directory

  • mkdir

  • Create multiple-level directories

    • mkdir -p

Create a new document

  • touch

View document in split screen

  • less

Move/Rename

  • mv

Copy

  • File

    • cp

  • Directory

    • cp -r

  • Overwrite and display prompt message

    • cp -i

  • Copy multiple files

    • cp

View document from the beginning

  • head -n 1

    • head -1

  • Display the first ten lines by default

View document from the end

  • tail -n 1

    • tail -1

  • Display the last ten lines by default

Output lines that contain a specified string

  • grep

  • Invert match

    • grep -v

  • Ignore case

    • grep -i

  • Lines that start with xxx

    • grep ^xxx

  • Lines that end with xxx

    • grep xxx $

Shut down

  • poweroff

Create a new user

  • useradd

Alias

  • Define alias

    • alias =”“

  • View alias

    • alias

  • Remove alias

    • unalias

Pipe symbol

  • |

Output text

  • echo

Create a user

  • useradd

  • User ID

    • useradd -u

  • Home directory path

    • useradd -d

  • Login interpreter

    • useradd -s

  • Additional group

    • useradd -G

View user information

  • id

Packaging (archiving and compressing)

  • tar-related operations

Remote login

  • ssh name:ip

  • Quit

    • exit

Directory Structure: Tree Structure

/: Root directory

..

  • Parent directory

.

  • Current directory

~

  • Home directory of the current logged-in user

/dev(ice): Stores device-related data

/root

  • Home directory of administrator root

/home

  • Stores home directories of all regular users

~user

  • Represents the home directory of user “user”

IDE: /dev/hd

SCSI: /dev/sd
