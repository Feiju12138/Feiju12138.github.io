Preface

Struts2 vulnerability exploitation scanning tool is a scanning and exploitation tool based on the publicly available Structs2 high-risk vulnerability exploits on the Internet. Currently supported vulnerabilities include: S2-001, S2-003, S2-005, S2-007, S2-008, S2-009, S2-012, S2-013, S2-015, S2-016, S2-019, S2-029, S2-032, S2-033, S2-037, S2-045, S2-046, S2-048, S2-052, S2-053, S2-devMode, S2-057 (Github)

This document is for learning purposes in network information defense

Identification of Sites Using Struts2 Framework

  • If the page suffix ends with .do or .action
  • If devMode is enabled, you can access the page /struts/webconsole.html
