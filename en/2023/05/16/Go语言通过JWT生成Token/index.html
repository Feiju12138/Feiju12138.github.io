Pitfalls

  • Error while generating or parsing the token: token signature is invalid: key is of invalid type

Reason 1

  • While generating or parsing the token, the key should be of type []byte, not string or other types.

Solution

  • The key should be of type []byte while generating or parsing the token.

Reason 2

  • The signing method should be jwt.SigningMethodHS256, not jwt.SigningMethodES256.
