Preface

iKuai deploys a TimeMachine server using the AFP protocol via Docker.

Preparations

  • iKuai can use Docker’s privileged mode

Link

Deploying a TimeMachine server using the AFP protocol

--net=: specifies Docker’s network
--ip=<ip>: specifies the IP address
/root: specifies the directory for storing MacOS backups

Backing up MacOS via the backup server

  • In Finder, go to Go->Connect to Server, and login to the file sharing server

Server: smb://<ip>
Username: timemachine
Password: timemachine

  • Go to System Preferences -> Time Machine -> Choose Backup Disk

  • Select the TimeMachine file sharing server as the TimeMachine backup server
