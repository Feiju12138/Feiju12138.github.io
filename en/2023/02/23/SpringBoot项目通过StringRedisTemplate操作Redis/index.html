Foreword

SpringBoot project operates Redis through StringRedisTemplate

Importing Dependencies

Getting Object

Adding Data

<key>: Key
<value>: Value

Overwriting with New Value

  • Returns the overwritten value

<new_value>: New value

Setting Expiration Time

  • Automatically destroyed when expiration time is reached

<num>: Specified expiration time, units determined by the fourth argument
TimeUnit.SECONDS: In seconds

Adding Data if Not Exist

  • If the key does not exist, insert the key-value pair; if the key exists, abort inserting data
  • Returns a boolean value indicating whether the insertion is successful

Batch Inserting Data

Overwriting Batch Insertion Data

  • If a key already exists, it is directly overwritten, and the insertion of all data continues

Non-Overwriting Batch Insertion Data

  • If a key already exists, the insertion is aborted and all data is rolled back
  • Returns a boolean value indicating whether the insertion is successful

Deleting Data

  • Remove the key-value pair by specifying the key
  • Returns a boolean value indicating whether the deletion is successful

Batch Deletion

  • Returns the number of successfully deleted data

Querying Data

  • Returns null if the value does not exist

<key>: Get the value by key

<str>: Substring

Processing Values of Numeric Type

  • If the key is not specified, the key-value pair is created immediately by the specified key, and the value is 0 by default
  • If the value corresponding to the specified key is non-numeric, an error is reported

<index_start>: Starting position index
<index_end>: Ending position index

Appending Data to the Set

  • Returns the length of the new set

<str>: Appended set subelement

Appending Data at the Specified Index Position
  • If the specified index position does not have data, add data
  • If the specified index position has data, overwrite the data
  • If the index is exceeded, an error is reported

<index>: Index position
<str>: Appended data

Prepending Data to the Set
  • If the key does not exist, the key-value pair is created immediately

Removing Data from the Set

  • Returns the removed data after removal
  • If the set is empty after removal, the key-value pair is directly deleted
Removing Data from the Head of the Set
Specifying Waiting Time
  • Specify the waiting time, if data appears within the waiting time, remove the data, if no data appears, no action is taken
Removing Data from the Tail of the Set
Specifying Waiting Time
  • Specify the waiting time, if data appears within the waiting time, remove the data, if no data appears, no action is taken
Removing Specified Values
  • Remove values that are the same as the specified string

<str>: String used for comparison
<num>: If there are multiple identical values, specify which to remove

>0: Remove the <num>th appearance of the same value from left to right
<0: Remove the <num>th appearance of the same value from right to left
=0: Remove all appearances of the same value
