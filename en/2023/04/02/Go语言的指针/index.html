Introduction

Notes on pointers in the Go language

Declaration and Definition of Pointers

Declaring a Pointer Variable

  • Declare a pointer, default is a null pointer (nil), with a default value of nil, value is 0
  • Accessing a null pointer will result in an error
  • 0~255 are reserved by the system and cannot be read or written to, attempting to do so will result in an error

data type: should correspond to the data type being stored

Defining a Pointer Variable

&variable: using the & operator, get the address of the variable and assign it to the pointer variable

Defining a Wild Pointer

  • The pointer variable points to an unknown memory address
  • Accessing a wild pointer will result in an error

Allocating Memory for a Pointer Variable

  • Allocate space in the heap, assign the address value to the pointer
  • The size of the memory space is the size of the specified data type, with a default value of the data type’s default value
  • In Go language, only allocate space is needed, no need to release space

Pointer Operations

  • Go language’s pointer type variables have the characteristic of efficient pointer access, while also avoiding pointer offsets and calculations, thus avoiding illegal modification of critical data

Taking the Address

  • Use &variable name to obtain the hexadecimal memory address value of the variable

Assigning Memory Address

  • Assign the memory address to store in the pointer

Dereferencing

  • Use *pointer variable to retrieve or modify the actual data pointed to by the pointer
