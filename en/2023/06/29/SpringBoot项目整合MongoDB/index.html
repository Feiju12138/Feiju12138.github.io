Add Configuration

<database_name>: Specify the database name
<username>: Username, can be omitted if MongoDB authentication is not enabled when starting the MongoDB service
<password>: Password, can be omitted if MongoDB authentication is not enabled when starting the MongoDB service

Create Entity Class

@Document(collection=""): Define the collection name. If the lowercase of the class name is the same as the collection name, the annotation can be omitted.
@Id: Define the primary key.
@Field(""): Define the field name. If the property name is the same as the field name, the annotation can be omitted.
@Indexed: Define a single field index.
@CompoundIndex(): Define a compound index.

Create Dao

User: The first parameter of the generic is the entity class.
String: The second parameter of the generic is the data type of the primary key.
