Preface

Automating the deployment of Hexo to Github Pages using Github Actions

Configure SSH Key Pair

  • Push the Hexo working directory and Hexo site directory to 2 Github repositories separately (Hexo working directory repository can be set as private)

Generate an SSH Key Pair

  • Generate an SSH key pair to deploy from the Hexo working directory Github repository hexo d to the Hexo site directory Github repository

Specify the Private Key in the Hexo Working Directory Github Repository


Specify the Public Key in the Hexo Site Directory Github Repository


Create Deployment Configuration

  • Create a directory to store the deployment configuration and create the deployment configuration file with any file name and the format Yaml
  • Write the configuration into the file

git config --global user.email "": Configure the email for your Git identity
git config --global user.name "": Configure the username for your Git identity

name: Specify the deployment configuration name
on: Specify the conditions that trigger the deployment

on.push.branches=["master"]: Specify to trigger the deployment when pushing to the master branch
on=["push"]: Specify to trigger the deployment on push operation

jobs: Specify the deployment tasks
HexoDeploy: Name of the deployment, customizable
runs-on: Starting container for the deployment
steps: Deployment steps
uses: Use deployment scripts from other Github repositories, actions is the official Github deployment scripts repository, where there are some commonly used deployment environments
name: Specific step name
secrets.: Variables in the Github repository Secrets
run: Shell commands executed in the current step, ‘|’ can be used to start defining each shell command on the next line without repeating - run

Advanced

  • Because the TCCLI tool can operate Tencent Cloud CDN products, caching refresh and cache preheating can be automated with Github Actions

Link

  • Automatically fetch all URLs mentioned in the sitemap.txt site map using a Python script, and perform Tencent Cloud CDN preheating on URLs other than articles in the list

Write an Automated Preheating Script

  • Before using, set site to your site root path, and set site_map to the site map path

Deploy Github Actions Configuration

  • Before using:
    • Set Tencent Cloud SecretId as the value of Hexo working directory repository’s Secret: TCCLI_API_SECRED_ID
    • Set Tencent Cloud SecretKey as the value of Hexo working directory repository’s Secret: TCCLI_API_SECRED_KEY

Tencent Cloud Secret retrieval link
Add Secret to Hexo working directory link

name: Deploy Hexo to GithubPage
on:
  push:
    branches: [ "master" ]
jobs:
  HexoDeploy:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        node-version: [18.x]
    steps:
    - uses: actions/checkout@v3
    - name: Use Node.js ${{ matrix.node-version }} environment
      uses: actions/setup-node@v3
      with:
        node-version: ${{ matrix.node-version }}
    - name: Configure SSH and Git environments
      env:
        ACTION_DEPLOY_KEY: ${{ secrets.HEXO_DEPLOY_KEY }}
      run: |
        mkdir -p ~/.ssh/
        echo "$ACTION_DEPLOY_KEY" > ~/.ssh/id_rsa
        chmod 700 ~/.ssh
        chmod 600 ~/.ssh/id_rsa
        ssh-keyscan github.com >> ~/.ssh/known_hosts
        git config --global user.email "feijuxiaogege@foxmail.com"
        git config --global user.name "FeiJu By Github Actions"
    - name: Configure Hexo environment
      run: |
        export TZ='Asia/Shanghai'
        npm install hexo-cli -g
    - name: Download Npm dependencies
      run: |
