Loading Alibaba iconfont
  • Take Douyin icon as an example
Download the icon
  • Log in to https://www.iconfont.cn/ and add the desired icon to the shopping cart, select the shopping cart, and choose Download Code


  • Create a fonts directory in the root directory of the project
  • Unzip the downloaded zip file and move the iconfont.json and iconfont.ttf to the fonts directory of the project

iconfont.json file is only used to find the value of the family field and the corresponding value of the icon’s unicode field. After use, it can be deleted and does not need to be kept all the time

Modify the configuration file
  • Load font static resources

family: Find the value of the family field in fonts/iconfont.json

Create a custom font class
  • Create an IconFont.dart in the lib directory

0xe8db: Find the value of the corresponding icon’s unicode field in fonts/iconfont.json, and add 0x as a prefix
fontFamily: Find the value of the corresponding icon’s family field in pubspec.yaml

Load third-party icons
  • Import the custom font class and load third-party icons through the static properties in the custom font class
