Create Smart Contract

  • Create a smart contract for storing user information

// SPDX-License-Identifier: GPL-3.0: Specify the open-source license of the contract
pragma solidity ^0.8.0;: Specify the compiler version the contract runs on
UserStorage: The contract name must match the file name

Create Deployment Script

  • Create a js file starting with a number in the migrations directory

Compile Smart Contract

  • After compiling, a build directory will be created to store the compiled files

Compile and Deploy Smart Contract

  • The migrate command compiles and deploys in sequence, including the functionality of the compile command
  • Compile and deploy the smart contract according to the deployment script
  • A deployment script will not be executed if it has been deployed once
  • To deploy a new smart contract, add a new deployment script file with a different name from the previously deployed smart contract script file
  • Deployment will consume gas fees upon success

Recompile and Redeploy Smart Contract

  • If the content of a script for a previously deployed smart contract has been modified without changing the script file name, the previous compilation results need to be overwritten and the smart contract redeployed using --reset
