【英文】Canvas学习笔记

Posted on 2021-03-24 Edited on 2024-03-28 Views:

Introduction

The Canvas tag can be used to draw graphics with JavaScript.

Creating a Canvas tag

  • Compatibility issue, only unsupported browsers will see the internal text in the Canvas tag, mainly targeting IE6, IE7, IE8

Properties

  • Canvas width and height
  • Do not use CSS to set the width and height, as using CSS will stretch the canvas, causing it to distort

Drawing context object

2d: There are two contexts, 2d and 3d

Pixelation of Canvas

  • When we use Canvas to draw a shape, the Canvas will pixelate the shape. The Canvas does not have the ability to change the content already on the canvas, which is why Canvas is lightweight.

Animation concept in Canvas

  • If we want the shape drawn in Canvas to move, we must program according to the logic of clear -> update -> render, in short, it is to redraw the shape

Implementing Canvas animation using object-oriented programming

Canvas API

Fill shape

Set color

  • Color must be set before drawing shapes

<color>: Color name

Fill rectangle

<x>: Horizontal coordinate of the shape from the origin (top left corner)
<y>: Vertical coordinate of the shape from the origin (top left corner)
<w>: Length of the rectangle
<h>: Width of the rectangle

Draw shape

Set color

Draw rectangle

  • Draw the border of the rectangle

Clear screen

<w>: Length range to clear
<h>: Width range to clear

Draw path

  • The purpose of drawing a path is to set an irregular polygon state. Paths are closed, and a certain sequence of steps must be followed when using paths for drawing.
  1. Set the starting point of the path
  2. Use drawing commands to draw the path
  3. Close the path
  4. Fill or draw the closed path shape
  • Automatic closure

For stroke, if closePath() is not used to close the drawing, the border will not automatically close
For fill, if closePath() is not used to close, the shape will automatically close

Arc

<x>: Horizontal coordinate of the center
<y>: Vertical coordinate of the center
<radius>: Radius
<startAngle>: Starting position. If it is a number, it represents the arc of the circle
<endAngle>: Ending position. If it is a number, it represents the arc of the circle. When the value is 2*Math.PI, 2*3.14, or 7, it represents a complete circle
anticlockwise: Clockwise or counterclockwise state, true for clockwise, false for counterclockwise

Set transparency

<num>: A number between 1 and 0, 1 for opaque, 0 for transparent

Line type

Set thickness

<num>: Thickness value, default is 1.0, representing 1 pixel

Set end shape

<value>

butt: Default value, end in square shape
round: End in round shape
square: End in square shape, but with an additional rectangle about 1/2 the thickness of the line in height

Set join shape

<value>

round: Join shape is rounded
bevel: Join shape is beveled
miter: Default value, join shape is a sharp angle

Set dashed line

<value>

[a,b]

a: Length of the line
b: Distance of gap

Starting offset

<value>: Offset value

Draw text

Set font

<size>: Font size, in pixels
<family>: Font name

Draw text

<str>: Text content
<x>: Horizontal position of the text
<x>: Vertical position of the text

Set text alignment

<value>

left: Align text to the left based on the x-axis

Canvas的API

填充图形

设置颜色

  • 颜色要先设置，再绘制图形

<color>: 颜色名称

填充矩形

<x>: 图形距离原点（左上角）的横坐标
<y>: 图形距离原点（左上角）的纵坐标
<w>: 矩形的长
<h>: 矩形的宽

绘制图形

设置颜色

绘制矩形

  • 绘制矩形边框

清屏

<w>: 清屏的长范围
<d>: 清屏的宽范围

绘制路径

  • 绘制路径目的是设置一个不规则的多边形状态，路径都是闭合的，使用路径进行绘制的时候需要既定的步骤
  1. 设置路径的起点
  2. 使用画图命令，画出路径
  3. 关闭封闭路径
  4. 填充或者绘制已经封闭路径的形状
  • 自动封闭

对于stroke，如果不使用closePath()封闭绘制，则边框不会自动封闭
对于fill，如果不使用closePath()封闭填充，则图形会自动封闭

圆弧

<x>: 圆心横坐标
<y>: 圆心纵坐标
<radius>: 半径
<startAngle>: 开始位置，如果是数字表示圆的弧度
<endAngle>: 结束位置，如果是数字表示圆的弧度，值为2*Math.PI、2*3.14、7时表示一个完整的圆
anticlockwise: 顺时针或逆时针状态，true表示顺时针，false表示逆时针

设置透明度

<num>: 1~0之间的数，1为不透明，0为透明

线型

设置粗细

<num>: 粗细值，默认为1.0，表示1像素

设置末端的形状

<value>

butt: 缺省值，末端以方形结束
round: 末端以圆形结束
square: 末端以方形结束，但是增加了一个__宽度是线段相同，高度是线段厚度一半__的矩形区域

设置相连部分的形状

<value>

round: 相连部分为圆角
bevel: 相连部分为平角
miter: 缺省值，相连部分为尖角

设置虚线

<value>

[a,b]

a: 线的长度
b: 空的距离

起始偏移量

<value>: 偏移量值

绘制文本

设置画笔

<size>: 字号，单位px
<family> : 字体名

绘制文本

<str>: 文本内容
<x>: 文本位置横坐标
<x>: 文本位置纵坐标

设置文本对齐方式

<value>

left: 基于x轴左对齐
