Vue component

Vue component structure

  • All Vue components are files with a .vue extension
  • App.vue is the root component of all components

template: structure
script: logic
style: style

Render data

  • The default export must be at the outermost level, and all Vue object listings must be listed within export default
  • In a Vue component, to prevent JavaScript data pollution, data cannot be an object but a function that returns an object

Import a component

  1. Import the component
  2. Register the component
  3. Use the component

Define custom functions

Native modifier

  • Native event listeners do not work in Vue components. To enable native event listeners, add the suffix .native after the native event listener

Parent-child component communication

  • The parent component passes a value to the child component using v-bind:custom-attribute
  • If the parent component passing value uses camel case, use - instead in the child component to receive the value using camel case
  • The child component receives the corresponding parameter through props and calls it directly in double curly braces
  • The values received by the child component can be arrays or objects

Parent component passing value to child component

Parent component passing value
Child component receives an array
  • If receiving as an array, there is no need to specify the type of the received data
Child component receives an object
  • If receiving as an object, you can restrict the type of the received data
Configure the parameter as a required prop

type: define the type
require: set it to true to enable the required value validation. If the value is not received, an error will be displayed in the console

Configure the default value for the parameter
  • If the default value is an object or an array, it needs to be defined as a function and return the default value

default: the default value if no value is received. Generally, do not mix with require

<template>
  <div>
    <h1>The value passed from the parent component (default value) is {{valueA.num}}</h1>
  </div>
</template>

<script>
export default {
  name: "Hello",
  props: {
    valueA: {
      type: Object,
      default: function() {
        return {
          num: 100
        }
      }
    }
  }
}
</script>
