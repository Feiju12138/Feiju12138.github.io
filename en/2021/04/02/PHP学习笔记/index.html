Preface

PHP (Hypertext Preprocessor) is an open-source general-purpose scripting language that is especially suitable for web development and can be embedded within HTML. PHP’s syntax borrows features from popular computer languages such as C, Java, and Perl, making it easy for developers to learn. The main goal of PHP is to enable web developers to write dynamic pages quickly, but PHP is also used in many other domains. (Wikipedia)

Variables

  • PHP does not require specifying variable types when defining variables. PHP variables can hold data of any type.
  • Rules for naming variables in PHP:
    • Start with $
    • Case-sensitive
    • Can only contain letters, numbers, and underscores
    • Cannot start with a number
    • Avoid using PHP keywords as variable names

When using echo to output, true is output as 1, and false is output as blank.

String

  • Strings can be defined using single quotes or double quotes. However, if you want to parse variables that appear in the string, double quotes must be used. Strings defined using single quotes cannot parse variables.
    • When parsing variables, the variable name must be an assigned variable.
    • If the variable name is followed by other strings, you can use a space to separate them.
    • If the variable name is immediately followed by other strings, you can use {} to wrap the variable name.
    • If you need to output the $ symbol in double quotes, you need to escape it with \.
  • Symbols in strings defined using single or double quotes need to be escaped with \. If there are a large number of symbols, you can use a heredoc syntax. All strings within the heredoc will be treated as is, without any modification. There should be no other characters (including spaces and tabs) before or after the heredoc HTML keywords.
