Preface

WeChat Mini Program, one type of mini program, also known as Wechat Mini Program in English, is an application that can be used without downloading or installing. It realizes the dream of “reachable touch” for applications, and users can open the application by scanning or searching. (Baidu Encyclopedia)

Registration

  • WeChat Official Account Platform

  • The email address should not be registered with WeChat Official Account Platform, WeChat Open Platform, or bound to a personal WeChat account.

Unit Conversion

  • rpx is a length unit unique to mini programs, which changes with the width of the screen.
  • In a 750px wide screen, the ratio between px and rpx is 1:1.
  • Formula: Screen width in rpx = 750rpx * Element width / Screen width in px.

Page Routing

  • Official Documentation

open-type="": Page jump method

navigateTo: Open a new page. Can only jump to non-TabBar pages (the current page is kept in the stack, and the new page enters the stack).
navigateBack: Page return. The url set for the return needs to be changed to the original page path. During the return, onLoad() will not be triggered because it has been saved in the page stack, but onShow() will be triggered (the current page is removed from the page stack).
switchTab: Open a new page. Can only jump to TabBar pages (all pages are removed from the page stack, and the TabBar page to be jumped to is kept).
reLaunch: Open a new page. Can jump to any page.
redirect: Page redirection. (The current page is popped out of the stack, and the new page is pushed into the stack).

url="": The path of the page to be jumped to, which can only be a relative path.
name="": Carries custom parameters, which can be called through the options property in onLoad().
