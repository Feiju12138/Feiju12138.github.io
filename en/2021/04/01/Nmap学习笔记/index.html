Preface

Nmap (Network Mapper) is a network security tool used for network discovery and security auditing. It is open source software. The name Nmap is a short form of Network Mapper. (Wikipedia)

This article is for learning purposes of network defense only.

Common Commands

  • Normal Scan: For regular users, use TCP scan for faster scanning but less secure.
  • Normal Scan: For ROOT users, use SYN scan for slower scanning but more secure.

Quick Scan

  • Includes IP address scanning and port scanning.
  • By default, only scans the first 1000 ports.

Host Discovery

  • Only includes IP address scanning, does not scan ports.

-sP, -sn: No port scanning.

  • Only includes port scanning.

Test Specific Requests

UDP

  • Scan for UDP services.

TCP Full Connect Scan

  • Performs a complete TCP three-way handshake during scanning to ensure TCP services are working properly.
SYN Half-connect Scan
  • Does not perform a complete TCP three-way handshake during scanning, only performs the second step.

Comprehensive Scan

  • Includes IP scanning, port scanning, and service version scanning.

OS Fingerprinting

  • View the operating system of the specified IP address and all the open port numbers on the system.
  • May not be accurate.
