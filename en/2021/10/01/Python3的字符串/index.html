Introduction

Notes on Python3 Strings

Concatenation

  • Use the plus operator to concatenate two strings. The data types on both sides of the plus operator cannot be non-string types when used for string concatenation.

<str_1>, <str_2>: strings

  • When the asterisk is used for string concatenation, the string on the left side of the asterisk is concatenated the number of times on the right side to get a new string.

<str>: string
<num>: number

Get a Range of Substrings

  • Get the substring with the range [<num_1>,<num_2>)
  • Specify the step

<step>: if it is a positive integer, step from left to right; if it is a negative integer, step from right to left

  • End at the end
  • If there is no step or the step is a positive integer, start from position 0 and end at position <num>
  • If the step is a negative integer, start from position -1 and end at position <num>

  • Start at -1

Searching

  • The find() function and the index() function can both search for substrings.
  • The difference between the find() function and the index() function is that:
    • The find() function returns the index if the substring exists; if the substring doesn’t exist, it returns -1.
    • The index() function returns the index if the substring exists; if the substring doesn’t exist, it raises an error.
  • The find() function and the index() function search for substrings from left to right, while the rfind() function and the rindex() function search for substrings from right to left.

str: a string variable

Setting String Alignment

  • The ljust() function can set the length of the original string and left-align it, returning the aligned string.
  • The rjust() function can set the length of the original string and right-align it, returning the aligned string.
  • The center() function can set the length of the original string and center-align it, returning the aligned string.
    • If the center() function cannot be perfectly centered, the right side will have 1 more padding character than the left side.

<len>: the length of the padded string
<s>: the string used for padding
