Preface

Python3 List Learning Notes

List Format

  • A list can store multiple data types at once, but usually stores data of the same data type.

Search

Element at a Specified Position

  • Access the element at a specified position directly through the index.

list: list variable
<index>: index position

Position of a Specified Element

  • The index function can find the index of the specified element, and will raise an error if it doesn’t exist.

<obj>: data in the list
<start>: starting index
<end>: ending index

Search for a Specified Element in a Range

Search for a Specified Element in the Entire List

Number of Occurrences of a Specified Element

  • The count function can count the number of occurrences of a specific element, and will return 0 if it doesn’t exist.

Length of the List

  • The len function returns the length of the list (number of elements in the list).

Check Existence

Check Existence

  • The ‘in’ keyword can check if a specified element exists in the list. It returns True if the element exists, else False.

Check Non-existence

  • The ‘not in’ keyword can check if a specified element does not exist in the list. It returns True if the element does not exist, else False.

Addition

Append Data to the End of the List

  • If a sequence is appended, the sequence object is directly appended to the end of the list.

<obj>: a data

Append One or Multiple Data to the End of the List

  • If a sequence is appended, the sequence data is unpacked, and each unpacked data is appended to the end of the list.

Add Data at a Specified Position in the List

  • Add data at a specified position in the list, shifting the other data back one position.

Deletion

Delete the Current List

  • Delete the entire list from memory.

Delete Data at a Specified Index

  • The ‘del’ keyword can directly delete the data at a specified index.
  • The pop function can delete the data at the specified index, and return the deleted data.

Delete Data at the End

  • If the index is not specified, the pop function will delete the data at the end of the list by default.

Delete Specified Data

Clear the List

  • Clear all data in the list, keeping an empty list.

Modification

Update Data at a Specified Index

<obj>: the new data to be updated

Reverse the Order of all Data in the List

  • The reverse function directly modifies the list by reversing the positions of all elements in the list.

Sort all Data in the List

key=<obj>: If sorting is based on a dictionary, you can specify the value of the key to sort by.
reverse=False: Default is False for ascending order. True for descending order.

Copy

  • The copy function can duplicate the list, returning a new list object.

list_new: the new list

Traversal

  • Access all data in the list in order.
  • For-loop is usually used to traverse all data.

Mathematical Operators

list1, list2: Any list variables
father: Parent list (refers to a list that contains another list as its element)
son: Child list (refers to a list that is contained by another list)

Intersection

  • Returns a list of the intersection of two lists.

Union

  • Returns a list of the union of two lists.

Complement

  • Also known as the difference

  • Also known as the relative complement

  • Returns a list of the complement of two lists.

Symmetric Complement

  • Removes the same elements in two sets and stores the distinct ones in a new set.

Superset

  • The operation results in a boolean value.
  • Determines whether one set has all the elements of another set.

Subset

  • The operation results in a boolean value.
  • Determines whether all elements of a set are completely contained in another set.
