Introduction

Notes on Python3 functions

Functionality of Functions

  • Integrating a independent code block into a whole and naming it as a function, so that it can be executed by calling the function name when needed

Defining a Function

Parameters of Functions

Positional Arguments

  • When calling a function, pass arguments based on the position of parameters defined in the function
  • If the number of arguments passed during the call does not match the number of parameters defined, an error will be raised

Keyword Arguments

  • When calling a function, pass arguments in the form of key=value

Default Arguments

  • When defining a function, set default values for parameters

Variable-length Arguments

  • Variable-length arguments are also known as variable arguments
  • Used in scenarios where the number of arguments to be passed is uncertain at the time of calling

Pack Positional Arguments

  • Accepts variable-length positional arguments
  • Parameter name is usually written as args
  • The received argument is a tuple data

Pack Keyword Arguments

  • Accepts variable-length keyword arguments
  • Parameter name is usually written as kwargs
  • The received argument is a dictionary data

Scope of Variables

  • Local variables: Variables inside a function can only be called within the function body
  • Global variables: Variables outside the function can be called in any function
  • If there are both local and global variables with the same name, the local variable is called
