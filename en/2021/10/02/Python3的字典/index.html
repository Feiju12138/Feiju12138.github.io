Preface

Python3 Dictionary Study Notes

Defining a Dictionary

  • Data is stored in the form of key-value pairs, with each pair separated by a ,

dic: dictionary variable

Adding or Modifying Key-Value Pairs

  • If the key does not exist, the data will be added
  • If the key exists, the data will be modified

<key>: key
<value>: value

Deleting a Dictionary

  • Use the del keyword or del function to delete a dictionary

Deleting Key-Value Pairs

  • The del keyword can be used to delete key-value pairs from a dictionary. If the pair exists, it will be deleted; if it does not exist, an error will be raised

Clearing a Dictionary

  • Clearing all key-value pairs in a dictionary, leaving an empty dictionary

Searching by Key

  • If the key exists, its corresponding value will be returned
  • If the key does not exist, an error will be raised

Searching by Function

Searching by Key

  • The get function can search for data in the dictionary by key. If the key exists, its corresponding value will be returned; if the key does not exist, a default value will be returned. If no default value is specified, None will be returned

<default>: default value, default is empty

Listing All Keys

  • The keys function can return all keys in the dictionary as an iterable sequence

Listing All Values

  • The values function can return all values in the dictionary as an iterable sequence

Listing All Key-Value Pairs

  • The items function can return all key-value pairs in the dictionary as an iterable sequence
