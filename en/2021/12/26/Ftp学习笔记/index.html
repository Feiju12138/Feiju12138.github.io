Set the Mode

  • The mode switching is mainly for preventing errors with the newline character \r\n on Windows.
    • If you want to automatically handle \r\n, i.e., when processing text files, it is recommended to use the ASCII mode.
    • If you do not want to handle \r\n, i.e., when processing non-text files, it is recommended to use the binary mode.

Switch to ASCII mode (default)

Upload a Single File

  • FTP does not support uploading folders.

<file_local>: Local file path (before uploading)
<file_remote>: Remote file path (after uploading)

Upload Multiple Files

  • Using mput can upload multiple files, but mput does not support absolute paths, so whether local or remote, switch to the path of the file (before/after uploading).

Basic Commands

Command for Operating the Remote Command for Operating the Local
lcd / cd / Switch directory
lpwd pwd View the current directory
lls ls View all files in the current directory
lmv <file_old> <file_new> mv <file_old> <file_new> Move files or directories
lrename <name_old> <name_new> rename <name_old> <name_new> Rename files or directories
lrm <file> rm <file> Delete files or directories
lmkdir <dir> mkdir <dir> Create a directory
lrmdir <dir> rmdir <dir> Remove a directory
