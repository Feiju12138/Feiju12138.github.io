Introduction

Python is a widely used interpreted, high-level, and general-purpose programming language. Python supports multiple programming paradigms, including functional, imperative, structured, object-oriented, and reflective programming. It has a dynamic type system, garbage collection capabilities, can automatically manage memory usage, and has a large and extensive standard library. (Wikipedia)

  • If it is a list representing a boolean value: Non-empty list is True, empty list is False
  • If it is a None value representing a boolean value: False

Logical Operations with Numbers

and
  • Returns 0 if there is a 0
  • Otherwise, returns the last non-zero number
or
  • Returns 0 if all are 0
  • Otherwise, returns the first non-zero number

Simplified Writing of Logical Operations

Bitwise Operators

  • &: Bitwise AND. Performs a bitwise AND of two numbers, 1 if both bits are 1, 0 otherwise
  • |: Bitwise OR. Performs a bitwise OR of two numbers, 0 if both bits are 0, 1 if any bit is 1
  • ^: Bitwise XOR. Performs a bitwise XOR of two numbers, 0 if the two bits are the same, 1 if the two bits are different
  • ~: Bitwise NOT. Performs a bitwise NOT of a number, 0 becomes 1, 1 becomes 0, and the first digit of the result becomes negative if it is 1
  • <<: Left Shift. Shifts the bits of the left-hand operand to the left by the number of positions specified in the right-hand operand, filling with 0. Each shift by 1 position is equivalent to multiplying the left-hand operand’s decimal number by 2
  • >>: Right Shift. Shifts the bits of the left-hand operand to the right by the number of positions specified in the right-hand operand, filling with 0. Each shift by 1 position is equivalent to dividing the left-hand operand’s decimal number by 2

Membership Operators

  • in, not in
  • Used to check if the second part contains the first part

Identity Operators

  • is, is not
  • Used to check if two objects reference the same object

Operator Precedence

  • Arithmetic Operators -> Bitwise Operators -> Comparison Operators -> Augmented Assignment Operators -> Identity Operators -> Membership Operators -> Logical Operators -> Assignment Operators
