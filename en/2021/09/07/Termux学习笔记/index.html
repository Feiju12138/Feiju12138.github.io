Posted on 2021-09-07 Edited on 2024-03-28 Views:

Foreword

Termux is an Android terminal emulator and Linux environment app that works directly with no rooting or setup required. A minimal base system is installed automatically - additional packages are available using the APT package manager (official website).

The initial startup requires automatic installation of the environment (may require going over the Great Firewall).

Changing Sources (Optional)

  • Changing to Tsinghua Mirror Source in China

Solution 1

  • Use the built-in GUI program to change the update source

Solution 2

  • Automatically change the update source through the script

Solution 3

  • Manually specify the update source (by editing the configuration file with the built-in program)
  • This method requires the installation of the vim editor if it is not already installed

$PREFIX/etc/apt/sources.list

Caveat: Although this method is recommended by the official website and many people have configured it this way, it does not work after testing. This may be because after updating the Termux version, three configuration files need to be configured, but this method can only edit one of the configuration files.

Solution 4

  • Manually specify the update source (by directly editing the configuration file)
  • This method requires the installation of the vim editor if it is not already installed

$PREFIX/etc/apt/sources.list

$PREFIX/etc/apt/sources.list.d/science.list

$PREFIX/etc/apt/sources.list.d/game.list
