Drawing

Draw Rectangle

canvas: window object
<x>: rectangle position x-coordinate
<y>: rectangle position y-coordinate
<weight>: rectangle width
<weight>: rectangle height
<line>: if not 0, it represents line width; if 0, it represents fill

Draw Circle

canvas: window object
<x>: circle position x-coordinate
<y>: circle position y-coordinate
<r>: circle radius
<line>: if not 0, it represents line width; if 0, it represents fill

Draw Arc

  • Since arc involves angles, the math package is usually introduced to calculate angles

canvas: window object
<x>: arc position x-coordinate
<y>: arc position y-coordinate
<width>: arc width
<width>: arc height
<angle_start>: arc start angle
<angle_end>: arc end angle
<line>: if not 0, it represents line width; if 0, it represents fill

Draw Ellipse

canvas: window object
<x>: ellipse position x-coordinate
<y>: ellipse position y-coordinate
<width>: ellipse width
<width>: ellipse height
<line>: if not 0, it represents line width; if 0, it represents fill

Draw Bezier Curve

  • aalines is used to draw anti-aliased lines
  • There is no direct function in Pygame to draw Bezier curves, so you can use the anti-aliased line function to create a curve with multiple 1-pixel circle points or rectangle points

False: not connected at the beginning and end
[[<x1>, <y1>], [<x2>, <y2>], ...]: list of points containing x and y coordinates
<line>: since it is drawing a line, it can only be line width
